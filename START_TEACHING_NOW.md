## ❓ 常見問題

**Q：看不到新增內容？**  
A：清除瀏覽器快取（Ctrl+Shift+Delete）→ 重新打開

**Q：Flask 報錯？**  
A：確保 Python 環境已配置，依賴已安裝：
```bash
pip install -r requirements.txt
```

**Q：我應該立即交易嗎？**  
A：否。建議先用紙筆記錄 1 週的候選股，不動真金，驗證信號準確度。

**Q：新增內容會改變掃描結果嗎？**  
A：不會。教學內容完全獨立於掃描邏輯。VCP 參數優化是另外進行的 (見 trader_config.py)。
