📅 每日工作流程 Daily Workflow

使用此程式進行 Qullamaggie 波段交易的標準流程：

步驟 1：開啟掃描頁面 Step 1: Open QM Scan

  • 內導航列點擊 「QM 掃描」 或訪問 /qm/scan
  • 這是您每日工作的起點，用來發現新的交易機會
  • 建議：每天市場收盤後運行一次，以獲得最新的股票設置

步驟 2：設置掃描參數 Step 2: Configure Scan Parameters

  • 最低星級 (Min Stars)
    • 推薦值：≥4★（星級越高，進場可能性越高）
    • 過濾基數：系統會自動過濾 3.5 分以下的股票
    • 激進者：可設為 3.5；保守者：設為 4.5-5★
  • 排名限制 (Top N)
    • 推薦值：50（顯示前 50 個星級最高的結果）
    • 高於 50 的值會讓結果過多，難於精選
    • 低於 30 的值可能遺漏機會，尤其在弱勢市場中

步驟 3：執行掃描 Step 3: Run Scan

  • 點擊 「運行掃描」 按鈕開始掃描全市場
  • 觀看進度指示欄，通常需要 30-120 秒完成（取決於網絡和計算機）
  • 等待轉盤停止和「掃描完成」提示出現
  • 系統會自動根據 QM 方法計算每支股票的星級評分

步驟 4：檢查結果 Step 4: Review Results

  • 如果顯示 「在熊市環境中沒有掃出好結果」 → 等待市場環境改善，暫時不進場
  • 結果表格按星級降序排列（5★ 最佳）
  • 關注 綠色行（5★）及 黃色行（4★）的股票
  • 灰色行（3★）和紅色行（<3★）通常被跳過

步驟 5：深度分析 Step 5: Deep Dive Analysis

  • 在結果表中點擊任何 股票代號（如 NVDA、TSLA）
  • 系統自動導航到 /qm/analyze?ticker=XXXX 進行該股詳細分析
  • 檢查該股的：星級計分、設置類型、技術指標、交易計畫

VET 徽章解析 VET Badge Explanation

若股票顯示 ❌ VET 紅徽章，表示觸發以下風險因素之一：

  • ADR <5% → 波動太小，無法產生足夠利潤
  • 熊市環境 → 市場環境不利，失敗機率高
  • 其他 VETO 條件 → 見 Qullamaggie 嚴格的入場規則

⚠️ 重要：Qullamaggie 的規則是嚴格的。若有 VET，除非您有深刻理由，否則應跳過。

6 維度雷達圖 6-Dimension Radar

分析頁顯示股票在 6 個不同維度的得分：

  • ①趨勢 Trend → 價格是否在主要均線上方，動能是否向上
  • ②設置 Setup Type → 當前出現的突破形態品質（HTF 最好）
  • ③動能 Momentum → 6 個月漲幅和 RS 相對強度排名
  • ④進場時機 Entry Timing → 現在是否是最佳進場點
  • ⑤風險/回報 Risk/Reward → 止損位與目標價的比例
  • ⑥交易量 Volume Profile → 成交量是否支持突破

🎯 越接近外圍六邊形，該維度表現越佳。

🎯 進場決策 Entry Decision Framework

進場清單 Entry Checklist

在進場前，檢查以下所有條件是否滿足：

✅ 必須全部綠燈 ALL GREEN
  • 星級 ≥ 4★（或有特殊催化劑時 3.5★）
  • 無 VET 紅徽章或風險過濾標記
  • ADR ≥ 5%（足夠波動，能產生利潤）
  • 市場環境 = 牛市或穩定（非熊市）
  • RS 排名 ≥ 70 百分位數（強於大盤）
  • 最近 6 個月漲幅 ≥ 150%（動能充足）
  • 設置類型品質良好（HTF > LADDER > 其他）
  • 你有足夠的資本和心理準備承受此風險

決策樹 Decision Tree

掃描結果 ↓

  1. 有 VET 紅徽章？ → YES → ❌ 跳過，除非你對自己的判斷非常有信心
  2. 星級 < 4★？ → YES → △ 考慮跳過，除非有特殊催化劑或設置非常清晰
  3. ADR < 5%？ → YES → ❌ 跳過，波動太小無法產生足夠風險/回報
  4. 市場環境 = 熊市？ → YES → ⚠️ 謹慎，勝率大幅下降
  5. 以上全部通過？ → YES → ✅ 準備進場，確認止損和頭寸大小

⚠️ 風險警示 Red Flags & When to Pass

停損系統 Minervini 3-Segment Stop System

Qullamaggie 使用嚴格的 3 段式止損系統，確保虧損有限：

階段 Stage 時間框架 止損邏輯 Stop Logic 目的 Purpose
第 1 段 Day 1 進場當日至第 2 日 固定於進場價下方 ADR 的 80% 快速評估突破有效性，最小化虧損
第 2 段 Day 2-3 第 2-3 日 逐漸上移至最新的搖晃低點 (Recent Swing Low) 保護利潤，隨著上漲動態調整
第 3 段 Day 3+ 第 3 日后持倉 跟隨移動平均線或突破的持續基礎 讓利潤跑動，直到基本面破壞

VETO 觸發條件 VETO Triggers

以下任何一個條件被觸發，該股票應該被 VETO（過濾），不應進場：

  • ❌ ADR < 5% → 波動太小，無法靠波動產生足夠利潤
  • ❌ 市場環境 = 熊市 → 環境不利，逆勢操作勝率低
  • ❌ RS 排名 < 70% → 股票相對大盤太弱，不值進場
  • ❌ 6M 漲幅 < 150% → 動能不足，不是 Qullamaggie 青睞的快速上升者
  • ❌ 設置位置不佳 → 不在漲幅最大處入場，代表太晚加入

❓ 常見問題 Frequently Asked Questions

「應該以 3.5★ 進場嗎？」Can I enter with 3.5 stars?

通常不建議。Qullamaggie 的方法對進場要求很高：

  • ✅ 如果設置類型是 HTF 且無 VET → 可考慮 3.5★
  • ❌ 如果沒有特殊理由 → 等待 4★+ 的候選
  • 通常 4★+ 的勝率更高，值得等待更優質的機會

「我可以覆蓋 VETO 嗎？」Can I override VETO?

不建議。VETO 條件代表：

  • 市場環境不利（熊市）→ 統計上失敗率高
  • 波動太小（ADR < 5%） → 無法產生足夠利潤
  • 相對強度太弱（RS < 70%） → 不跟隨大盤趨勢

Qullamaggie 強調嚴守紀律。犯規越少，長期勝率越高。

「應該多常掃描？」How often should I scan?

建議：每個交易日的市場收盤後運行一次。

  • ✅ 每日掃描 → 抓住新的突破機會，最實時
  • △ 每周掃描（周五） → 可以，但可能遺漏周中的機會
  • ❌ 不定期掃描 → 容易遺漏機會繁整個缺口

「為什麼某些股票消失了？」Why did a stock disappear from results?

股票可能因以下原因從掃描結果消失：

  • 突破失敗 → 股票跌破設置，被篩除出去
  • 波動降低 → ADR 跌破 5% 閾值
  • 市場環境變化 → 市場進入熊市，篩選變嚴格
  • RS 排名下降 → 相對大盤表現轉弱

這是正常的。Qullamaggie 的方法動態地篩選候選，確保只關注最好的設置。

「進場後應該如何操？」What should I do after entry?

  1. Day 1：嚴格遵守 Day 1 止損，監控突破有效性
  2. Day 2-3：若上漲，逐日上移止損；若振盪，觀察確認
  3. Day 3+： 讓利潤跑動，使用移動平均線或者關鍵支撐作為動態止損
  4. 何時出場：
    • ✅ 當移動平均線被破壞時 → 基本面已改變
    • ✅ 達到目標價時 → 通常獲利足夠，可考慮部份出場
    • ✅ 止損被擊中 → 嚴格遵守止損，保護資金

「為什麼我的利潤和目標價不一致？」Why is my profit different from target price?

  • 目標價是基於當時的技術面判斷。
  • 市場可能超預期上升（更好）或提前下跌（更差）
  • 實際獲利取決於你的進場點、頭寸規模和出場點
  • 重要的是保護止損並讓利潤跑動
