

# Martin Luk vs. Kristjan Kullamägi — 買入時機深度比較

> 以下整理來自兩位交易員的公開訪談、社交媒體與直播內容。Kristjan 的資料較完整（大量 Twitter/訪談），Martin 的部分則綜合自他已分享的公開內容，若有細節出入歡迎指正。

## 一、總覽：入場哲學的根本差異

| 維度 | Martin Luk | Kristjan Kullamägi |
|------|-----------|-------------------|
| **主要持倉週期** | 偏短線波段（數天），也做日內 | 波段為主（數天～數週） |
| **核心入場邏輯** | 等待盤中「確認訊號」後才出手；重視 intraday price action | 以日線圖形態成熟為準，盤中只做執行 |
| **時區** | 香港（美股 21:30–04:00 HKT 夏令） | 瑞典（美股 15:30–22:00 CET） |
| **對盤前的態度** | 高度重視盤前 volume/price，會據此決定要不要參與 | 盤前做掃描與計劃，但入場決策幾乎全在盤中 |
| **入場次數** | 較頻繁，願意同日多次進出 | 精選，寧可錯過也不追 |

## 二、Kristjan Kullamägi 的入場時機

Kristjan 公開說過他有兩大入場模式：**日線形態突破（Pattern Breakout）** 和 **事件性跳空（Episodic Pivot, EP）**。

### 2.1 盤前階段（Pre-market）—— 只做計劃，不做交易

Kristjan 的盤前流程：

```
歐洲下午 ~14:00 CET（美股開盤前 1.5 小時）
├── 檢查 watchlist 上的股票是否觸及 pivot（突破點）
├── 掃描盤前跳空股（gap scanner）篩選 EP 候選
├── 設定 alert 在關鍵價位
└── 決定今天最多 2-3 隻重點標的
```

> **他極少在盤前直接下單買入。** 原因是盤前流動性差，容易被假突破欺騙。

### 2.2 入場模式一：日線形態突破（VCP / Flat Base / Cup-with-Handle）

**核心原則：在 pivot point 被突破的「那一刻」買入。**

**Pivot Point 的定義：**
- VCP（Volatility Contraction Pattern）：最後一個收縮區間的高點
- 平坦底部（Flat Base）：底部整理區的上軌
- 杯柄形態（Cup-with-Handle）：柄部的高點

**入場條件清單：**

```
✅ 股價在 Stage 2 上升趨勢（200MA 上升，股價在 200MA 之上）
✅ 50MA 在 200MA 之上
✅ RS line（相對強度線）接近或創新高
✅ 底部整理至少 3-5 週
✅ 波動性逐漸收縮（後面的回調幅度 < 前面的）
✅ 突破當日成交量 ≥ 1.5 倍 50 日平均量
✅ 股價穿越 pivot price
```

**買入位置的精確規則：**

```
買入價位 = Pivot Price 至 Pivot Price × 1.05 之間
（即不追超過 pivot 5% 以上的股票）
```

**例子：**
假設股票 ABCD：
- 形成 VCP，最後收縮區高點 = $50.00（這就是 pivot）
- 50 日平均成交量 = 100 萬股

```
入場條件觸發：
  價格突破 $50.00 ✓
  當日至突破時刻累計量 > 75 萬股（按比例推算全日可達 150 萬+） ✓
  
買入區間：$50.00 ~ $52.50（$50 × 1.05）
超過 $52.50 → 放棄，不追

止損設在 VCP 最後收縮低點，例如 $47.00
風險 = ($50.00 - $47.00) / $50.00 = 6%
```

**日內時段偏好：**

```
最佳入場時段：開盤後 30 分鐘 ~ 2 小時（09:30-11:30 EST）
  → Kristjan 說過「最強的突破在早盤就會發生」
  → 早盤突破伴隨大量 = 機構在進場

次佳：午盤後（13:00-14:00 EST）
  → 有時突破在午盤後才發動

避免：開盤前 5 分鐘，以及收盤前 30 分鐘
  → 開盤頭 5 分鐘波動太大，容易是假突破
  → 尾盤突破通常缺乏跟進動力
```

### 2.3 入場模式二：事件性跳空（Episodic Pivot, EP）

EP 就是因為重大消息（財報、FDA 批准、合約等）而大幅跳空高開的股票。

**Kristjan 的 EP 入場有兩種做法：**

#### 做法 A：當日跳空買入（較激進）

```
條件：
  ✅ 跳空幅度 ≥ 10%（他偏好更大的 gap）
  ✅ 盤前成交量異常巨大（至少 5-10 倍平均盤前量）
  ✅ 催化劑是「真正改變公司基本面」的事件（如盈利大幅超預期）
  ✅ 開盤後股價沒有立刻大幅回落填補缺口

入場時機：開盤後 5-15 分鐘，觀察第一根 5 分鐘 K 線
  → 若開高後守住盤前低點 → 買入
  → 若開高後直接灌破盤前低點 → 放棄
  
止損：設在 gap day 的低點之下
```

#### 做法 B：等 EP 後短暫整理再買突破（較保守，他更常用）

```
EP 發生 → 等待 1-5 個交易日
  → 股價在跳空後的高位窄幅整理（形成 flag 或 tight range）
  → 突破這個短期整理的高點 = 入場

例子：
  Day 0: 股票因財報從 $30 跳空到 $40，收 $38
  Day 1-3: 股價在 $37-$40 之間窄幅整理
  Day 4: 突破 $40.00 → 買入
  止損: $36.50（整理區低點下方）
```

### 2.4 Kristjan 入場時機的核心公式

**倉位大小公式：**

$$Position\ Size = \frac{Portfolio \times Risk\%}{Entry\ Price - Stop\ Price}$$

**例子：**
```
帳戶 = $500,000
每筆交易風險 = 0.5%（$2,500）
Entry = $50.00
Stop = $47.00
Risk per share = $3.00

Position Size = $2,500 / $3.00 = 833 股
Position Value = 833 × $50 = $41,650（約帳戶的 8.3%）
```

**成交量確認公式：**

$$Volume\ Ratio = \frac{Breakout\ Day\ Volume}{50\text{-}day\ Avg\ Volume}$$

```
Volume Ratio ≥ 1.5 → 可接受
Volume Ratio ≥ 2.0 → 理想
Volume Ratio < 1.2 → 可能是假突破，謹慎
```

## 三、Martin Luk 的入場時機

Martin 的風格在幾個關鍵點上跟 Kristjan 有顯著差異。

### 3.1 盤前階段 — 已經在做入場判斷

由於 Martin 在香港（美股開盤是晚上 9:30），他的盤前準備實際上佔據了他**整個下午和傍晚**：

```
香港下午 ~15:00-18:00（美股盤前數小時）
├── 做基本面/催化劑研究（看盤前有什麼消息）
├── 掃描盤前跳空股（重點看 gap% + 盤前量）
├── 標記關鍵價位：前日高低、盤前高低、重要均線
└── 建立初步計劃：什麼價位買、什麼量確認、止損在哪

香港晚上 ~20:00-21:30（美股開盤前 1-1.5 小時）
├── 觀察盤前 price action 的演變
├── 如果盤前量和價格動作非常強 → 可能在盤前試探性小量買入
└── 根據盤前表現調整計劃
```

**關鍵差異：Martin 有時會在盤前就小量建倉**，尤其當：
- 催化劑非常明確且強力（如財報大超預期）
- 盤前量已經非常大（表示有足夠流動性）
- 他判斷開盤後可能直接跳高，不給好的買入機會

而 Kristjan 幾乎不在盤前交易。

### 3.2 入場模式：更重視日內 Price Action 確認

Martin 跟 Kristjan 最大的差異之一：**Martin 更依賴日內圖（1 分鐘、5 分鐘 K 線）來確認入場點。**

#### 模式 A：開盤區間突破（Opening Range Breakout, ORB）

```
開盤後前 5-15 分鐘：觀察，不交易
  → 記錄這段時間的最高價和最低價 = Opening Range（OR）

入場條件：
  ✅ 股價突破 OR 高點
  ✅ 突破時成交量放大
  ✅ 股價在 VWAP 之上
  ✅ 大盤（SPY/QQQ）沒有強烈走弱

買入：OR 高點之上 $0.05-0.10
止損：OR 低點之下，或 VWAP 之下

例子：
  股票 XYZ 因財報跳空高開至 $25.00
  09:30-09:45 範圍：$24.50 - $25.80（Opening Range）
  09:46 突破 $25.80 + 量放大 → 買入 $25.85
  止損 $24.40（OR 低點下方）
```

#### 模式 B：盤前高點突破

```
記錄盤前最高價（Pre-market High, PMH）

入場條件：
  ✅ 開盤後股價回測但不跌破 VWAP
  ✅ 然後反彈突破 PMH
  ✅ 突破 PMH 時量放大
  
買入：PMH + $0.05-0.10
止損：VWAP 之下 或 開盤後低點之下

例子：
  股票 XYZ 盤前最高 $26.00
  開盤後回落到 $25.20（接近 VWAP $25.10）
  10:15 反彈突破 $26.00 + 量增 → 買入 $26.10
  止損 $25.00
```

#### 模式 C：VWAP 回踩買入

```
適用於已經確認有強催化劑的股票

觀察：開盤後股價快速上漲
等待：回踩到 VWAP 附近
確認：在 VWAP 處出現支撐（如 doji、hammer K 線）
買入：VWAP 反彈確認後
止損：VWAP 之下固定金額（如 2-3%）
```

### 3.3 Martin 的入場時段偏好

```
最活躍時段：開盤後 30-90 分鐘（09:30-11:00 EST）
  → 這是他的主戰場
  → 因為香港時間晚上 9:30-11:00，他還精力充沛

次要時段：午盤反轉（11:30-13:00 EST）
  → 如果早盤錯過，會看午盤是否有第二次機會

較少交易：下午盤（14:00-16:00 EST）
  → 香港時間已是凌晨 2-4 點
  → 除非持倉管理需要，否則傾向不開新倉
```

### 3.4 Martin 的成交量觀察方式

Martin 似乎更重視 **相對於日內時段的成交量**（volume vs. intraday expectations），而不僅僅是跟 50 日均量比較：

```
日內成交量確認：
  開盤後 30 分鐘的成交量 ≈ 全日量的 20-30%（正常市場）
  
  若頭 30 分鐘成交量已達到 50 日均量的 50% 以上
  → 表示今天成交量可能是正常日的 2-3 倍
  → 確認有足夠的參與者在驅動這個走勢

公式：
  Projected Daily Volume = (Volume in first 30 min) / 0.25
  
  若 Projected Daily Volume > 2 × 50-day Avg Volume → 強確認
```

## 四、核心差異對比

### 差異 1：入場的時間精度

```
Kristjan：「日線級別的精度」
  → 今天這根日 K 線突破了 pivot，我就買
  → 具體在日內什麼時間買入不是最重要的
  → 看的是日線形態是否成熟

Martin：「分鐘級別的精度」
  → 就算日線形態看起來好，也要等日內出現特定 pattern
  → 例如：ORB 確認、VWAP 回踩確認、PMH 突破確認
  → 什麼時間點買入很重要
```

### 差異 2：對「突破」的理解

```
Kristjan 的「突破」：
  → 日線收盤價突破 pivot，且成交量放大
  → 他可以接受日內暫時跌破 pivot，只要收盤站上
  → 評判標準是「這根日 K 線」

Martin 的「突破」：
  → 日內某個關鍵水平被突破（OR High、PMH、日內阻力）
  → 需要即時的成交量確認
  → 評判標準是「這根 5 分鐘 K 線」
```

### 差異 3：止損的設置邏輯

```
Kristjan：
  止損 = 日線形態的結構性低點
  止損距離通常 5-10%
  比較寬，但倉位相應較小
  
  Risk per trade = 0.25% ~ 1.0% of portfolio
  
Martin：
  止損 = 日內關鍵水平（VWAP、OR Low、PMH 下方）
  止損距離通常 2-5%
  比較窄，倉位相應可以較大
  
  Risk per trade = 0.5% ~ 1.5% of portfolio（推測）
```

### 差異 4：分批建倉 vs. 一次到位

```
Kristjan：
  傾向一次建滿倉位
  → 在 pivot 突破時買入完整倉位
  → 有時會用「加碼買入」：第一筆 50-75%，確認後加 25-50%
  → 但基本上是在 pivot 附近完成建倉

Martin：
  更傾向分批進場
  → 盤前小量試探（如總計劃倉位的 20-30%）
  → 開盤後確認加碼（再加 30-40%）
  → ORB 或 PMH 突破時加滿（最後 30-40%）
  
  這種分批策略降低了單一時點判斷錯誤的風險
```

### 差異 5：對大盤的敏感度

```
Kristjan：
  → 主要看個股形態，大盤只作為背景參考
  → 在確認的上升趨勢市場中更積極
  → 但不會因為大盤某天弱就放棄一個完美突破

Martin：
  → 更重視當天大盤的即時走勢
  → 若 SPY/QQQ 在開盤後走弱，他會延遲入場或降低倉位
  → 他可能有一個「大盤過濾器」：
  
  IF SPY < 今日 VWAP AND SPY < 昨日收盤
    THEN 減少新倉位 50% 或暫不入場
  ELSE
    正常執行計劃
```

## 五、並排實例對比

### 場景：某股票 MNOP 盈利後跳空高開

```
背景：
  前日收盤 $80.00
  盈利後跳空高開至 $96.00（+20%）
  盤前量巨大
  50 日平均量 = 200 萬股
  盤前已成交 300 萬股
  盤前高 = $98.00
  盤前低 = $93.50
```

**Kristjan 的做法：**
```
盤前：
  看到 EP 候選 → 放入當日觀察名單
  設 alert：$98.00（盤前高）、$100.00（心理關口）
  
開盤後：
  09:30 開盤 $96.50
  09:30-09:35 波動劇烈，在 $94-$97 之間震盪
  
  → Kristjan 等待
  → 他可能選擇「做法 B」：等 1-3 天短暫整理後再買
  
  如果選擇當日入場：
  09:45 股價站穩 $97，量持續放大
  09:50 突破 $98.00（盤前高）→ 考慮買入
  
  買入價：$98.20
  止損：$93.00（gap day 低點下方）
  風險：($98.20 - $93.00) / $98.20 = 5.3%
  
  Portfolio = $1,000,000
  Risk = 0.5% = $5,000
  Position Size = $5,000 / $5.20 = 961 股
  Position Value = 961 × $98.20 = $94,370（~9.4% of portfolio）
  
  或者更可能：
  → 等待 Day 1-3 整理
  → Day 2: 股價在 $95-$99 整理
  → Day 3: 突破 $99 → 買入
  → 止損 $94（整理低點）
```

**Martin 的做法：**
```
盤前（香港時間 ~20:00-21:30）：
  看到 EP 候選 → 分析催化劑品質
  盤前量已達 300 萬股（1.5 倍日均量）→ 非常活躍
  
  21:00（開盤前 30 分鐘）：
  盤前穩定在 $96-$97 → 決定參與
  可能盤前小量買入 200 股 @ $96.50（試探倉位）
  
開盤後：
  09:30 開盤 $96.50
  09:30-09:45 觀察 Opening Range
  OR: $94.50(L) - $97.80(H)
  
  09:47 股價突破 $97.80（OR High）+ 1 分鐘量放大
  → 加碼買入 500 股 @ $97.90
  
  10:05 股價突破 $98.00（PMH）
  → 再加碼 300 股 @ $98.10
  
  總持倉：1,000 股
  平均成本 = (200×96.50 + 500×97.90 + 300×98.10) / 1000
           = $97.67
  止損：VWAP（假設 $95.50）之下 → $95.30
  風險：($97.67 - $95.30) / $97.67 = 2.4%
```

**關鍵差異一目了然：**

| 項目 | Kristjan | Martin |
|-----|----------|--------|
| 盤前是否交易 | 否 | 可能小量試探 |
| 等待時間 | 可能等數天 | 開盤後 15-30 分鐘內決定 |
| 入場次數 | 1 次 | 2-3 次分批 |
| 止損依據 | 日線結構低點 | 日內關鍵水平（VWAP/OR Low） |
| 止損幅度 | ~5% | ~2-3% |
| 使用圖表 | 日線為主 | 1-5 分鐘為主 |

## 六、相關公式匯總

### 公式 1：倉位大小（兩人通用）

$$Shares = \frac{Account \times MaxRisk\%}{Entry - Stop}$$

### 公式 2：Kristjan 的最大可追價格

$$Max\ Buy\ Price = Pivot \times 1.05$$

> 超過此價不追。Kristjan 曾多次強調 "don't chase"。

### 公式 3：Martin 的日內成交量推算

$$Projected\ Volume = \frac{Volume_{first\ N\ min}}{\text{Typical \% of daily volume in first N min}}$$

常用近似值：

| 前 N 分鐘 | 佔全日量比例（近似） |
|-----------|-------------------|
| 前 5 分鐘 | ~8-10% |
| 前 15 分鐘 | ~15-18% |
| 前 30 分鐘 | ~22-28% |
| 前 60 分鐘 | ~35-40% |

```
例：前 30 分鐘成交 150 萬股
Projected Daily Volume ≈ 1,500,000 / 0.25 = 6,000,000
若 50-day avg = 2,000,000
Volume Ratio = 6,000,000 / 2,000,000 = 3.0 → 非常強的確認
```

### 公式 4：R 倍數（兩人通用，但計算基礎不同）

$$R = \frac{Entry - Stop}{Entry} \quad (\text{初始風險比例})$$

$$Reward : Risk = \frac{Target - Entry}{Entry - Stop}$$

```
Kristjan 典型：
  R = 5-10%（較寬止損）
  目標 R:R ≥ 3:1（讓利潤奔跑數週）

Martin 典型：
  R = 2-4%（較窄止損）
  目標 R:R ≥ 2:1（較快獲利了結）
```

### 公式 5：波動性收縮確認（Kristjan 的 VCP 驗證）

$$Contraction\ Ratio = \frac{後一次回調幅度}{前一次回調幅度}$$

```
理想情況：
  第 1 次回調：-25%
  第 2 次回調：-12%  → Ratio = 12/25 = 0.48
  第 3 次回調：-5%   → Ratio = 5/12 = 0.42

每次 Ratio < 0.6 → 波動性在收縮 → 形態健康
連續 2-3 次收縮後突破 → 就是 Kristjan 的入場點
```
