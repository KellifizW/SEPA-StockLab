### 1.1 Minervini 的交易哲學——十大核心信條

```
═══════════════════════════════════════════════════════
            📜 MINERVINI 十大核心信條
═══════════════════════════════════════════════════════

信條 1：保護資本永遠是第一優先級
─────────────────────────────────
• 你無法在市場中賺到錢如果你已經沒有資本
• 每一塊錢都是你未來賺更多錢的種子
• 虧損 50% 需要漲 100% 才能回本——
  這個非對稱性是你永遠必須尊重的數學事實
• 所有的規則、所有的紀律，最終都服務於這一條

信條 2：交易是概率遊戲，不是確定性遊戲
─────────────────────────────────
• 沒有任何系統能保證每筆交易都盈利
• 你的優勢來自於大量重複一個正期望值的系統
• 單筆交易的結果不重要——
  100 筆交易的總體結果才重要
• 接受不確定性是成為專業交易者的前提

信條 3：虧損是經營成本，不是失敗
─────────────────────────────────
• 開餐廳要付房租，做交易要付虧損
• 你的工作不是消除虧損，而是把虧損控制在小範圍內
• 小虧損 + 大獲利 = 長期盈利
• 一筆遵守規則的虧損交易 = 好的交易

信條 4：讓利潤奔跑，截斷虧損
─────────────────────────────────
• 這是所有成功交易者共同遵守的黃金法則
• 你的少數幾筆大贏的交易會貢獻你大部分的利潤
• 過早賣出贏家和過晚賣出輸家是最致命的兩個錯誤
• 你的盈虧比比你的勝率更重要

信條 5：只在有利的市場環境中交易
─────────────────────────────────
• 大盤方向是所有個股交易的背景
• 在熊市中做多就像逆流游泳——即使你很強壯也會被沖走
• 知道什麼時候不交易和知道什麼時候交易同等重要
• 持有現金是一種積極的策略選擇

信條 6：過程比結果重要
─────────────────────────────────
• 遵守規則但虧損 = 好的交易 ✅
• 違反規則但獲利 = 壞的交易 ⛔
• 專注於你可以控制的事情（你的決策和紀律）
• 讓概率處理你無法控制的事情（市場的走向）

信條 7：你必須完全為自己的結果負責
─────────────────────────────────
• 沒有人強迫你買入或賣出
• 市場不在針對你
• 歸咎於外部因素 = 放棄改善的力量
• 把責任放在自己身上 = 把改善的力量放在自己手中

信條 8：紀律是最大的競爭優勢
─────────────────────────────────
• 知識是必要條件，紀律是充分條件
• 在交易中，知道做什麼只值 10 分，
  實際去做值 90 分
• 紀律不是天生的——它是通過慣例和系統建立的
• 你的交易表現不會超過你的紀律水平

信條 9：一致性勝過爆發力
─────────────────────────────────
• 穩定的年均 20-25% 遠勝過大起大落的 +60%/-40%
• 回撤是複利的最大敵人
• 追求可重複的、可持續的過程
• 時間 + 一致性 = 複利的威力

信條 10：交易是一場馬拉松，不是短跑
─────────────────────────────────
• 不要追求在第一年就「成功」
• 給自己時間學習、犯錯、成長
• 每天正確地執行過程 = 每天的成功
• 5-10 年的持續改進 = 卓越的交易生涯
```

### 3.1 趨勢模板（Trend Template）—— 八大必要條件

```
═══════════════════════════════════════════════════════
         📐 趨勢模板（Trend Template）
         所有條件必須同時滿足
═══════════════════════════════════════════════════════

□ TT1: 股價 > 150 日均線（SMA 150）
□ TT2: 股價 > 200 日均線（SMA 200）
□ TT3: 150 日均線 > 200 日均線
□ TT4: 200 日均線已上行至少 1 個月
       （最好已上行 4-5 個月以上）
□ TT5: 50 日均線 > 150 日均線 AND 50 日均線 > 200 日均線
□ TT6: 股價 > 50 日均線
□ TT7: 股價高於 52 週低點至少 25%
       （很多最好的股票高於低點 100% 以上）
□ TT8: 股價在 52 週高點的 25% 以內
       （理想是在 15% 以內——越接近高點越好）

額外條件（非絕對但強烈建議）：
□ TT9: 相對強度（RS）排名 ≥ 70（理想 ≥ 80）
□ TT10: 屬於相對強勢的行業板塊

═══════════════════════════════════════════════════════

趨勢模板的意義：
• 確保股票處於 Stage 2 上升趨勢中
• Stage 2 是唯一應該買入和持有的階段
• 不符合趨勢模板的股票 = 立即排除，不做進一步分析
```

### 3.3 基本面篩選標準

```
═══════════════════════════════════════════════════════
         📊 基本面篩選標準
═══════════════════════════════════════════════════════

【盈利標準】
□ F1:  最近一季 EPS 同比增長 ≥ 25%
□ F2:  最近兩季 EPS 增長趨勢為加速
       （例如：Q1 +20% → Q2 +35% → Q3 +50%）
□ F3:  年度 EPS 增長 ≥ 25%
□ F4:  盈利是高質量的（非一次性因素）
       → 不是來自資產出售、稅務優惠等
       → 來自核心業務的持續性收入

【營收標準】
□ F5:  營收增長 ≥ 20%（或有加速趨勢）
□ F6:  營收增長應與盈利增長相匹配
       → 只有盈利增長而無營收增長 = 不可持續
       → 可能只是靠削減成本，而非真正的業務增長

【利潤率和效率標準】
□ F7:  利潤率（Net Margin）改善或處於高水平
□ F8:  ROE（股東權益回報率）≥ 17%
□ F9:  利潤率趨勢向上（而非向下）

【額外正面因素】
□ F10: 盈利超出分析師預期（Earnings Surprise）
□ F11: 分析師上調盈利預期（Estimate Revisions Up）
□ F12: 機構持股增加（連續數季）
□ F13: 有清晰的增長故事（催化劑）

【基本面紅旗（任何一個出現都要警惕）】
□ ⛔ 盈利增長減速（從 +40% 降到 +15%）
□ ⛔ 營收增長停滯但盈利靠削減成本維持
□ ⛔ 利潤率持續下降
□ ⛔ 應收賬款增長速度遠快於營收增長
□ ⛔ 盈利來自一次性因素

═══════════════════════════════════════════════════════
```

### 4.1 VCP（波動收縮型態）核心原理

```
═══════════════════════════════════════════════════════
         🔬 VCP 核心原理
═══════════════════════════════════════════════════════

定義：
VCP = 基底形成過程中，股價的波動幅度和成交量
     同時、逐步收縮的現象

供需邏輯：
每一次收縮 → 弱手被洗出 → 供給減少
→ 回調幅度更小 → 成交量更低
→ 最終供給枯竭 → 只需少量新買盤 → 突破

收縮序列示例：
• T1（第一次收縮）：回調 -25%，成交量高
• T2（第二次收縮）：回調 -15%，成交量中
• T3（第三次收縮）：回調 -8%，成交量低
• T4（第四次收縮）：回調 -3%，成交量極低（枯竭量）
• 突破：成交量暴增 → 價格快速上漲

═══════════════════════════════════════════════════════
```

---

### 4.2 基底品質評估標準

```
═══════════════════════════════════════════════════════
         📏 基底品質評估標準
═══════════════════════════════════════════════════════

【深度（Base Depth）】
□ ✅ 正常：12-35%
□ ⭐ 理想：15-30%
□ ⚠️ 過淺：<10%（可能只是短暫停頓）
□ ⛔ 過深：>40-50%（套牢盤壓力大、可能有基本面問題）
□ 例外：嚴重熊市中好股票可回調 40-50%，可接受

【寬度（Base Width / Duration）】
□ ⛔ 太短：<3-4 週（供給未充分吸收）
□ ⚠️ 邊緣：4-5 週
□ ✅ 正常：6-16 週
□ ⭐ 理想：8-12 週
□ ⚠️ 過長：>6-9 個月（可能失去動能）

【VCP 收縮（必須展示）】
□ 能識別出至少 2-3 次收縮（T-2 或 T-3+）
□ 每次收縮幅度逐步減小
□ 每次收縮成交量逐步降低
□ 最後一次收縮幅度 <10%（理想 <5%）
□ 最後一次收縮成交量 = 枯竭量

【成交量型態】
□ 基底左側到右側：成交量趨勢性下降
□ 基底右側出現枯竭量
□ 沒有在基底右側出現異常放量下跌

【基底右側 vs 左側】
□ 右側強於左側
□ 收盤價傾向於基底範圍的上半部
□ 上漲日多於下跌日

【其他品質因素】
□ 處於 Stage 2 上升趨勢中
□ 相對強度在基底期間維持高水平
□ 基底型態清晰可辨，不雜亂
□ 支點清晰可識別

═══════════════════════════════════════════════════════
```

### 6.2 止損規則

```
═══════════════════════════════════════════════════════
         ⛔ 止損規則（不可妥協）
═══════════════════════════════════════════════════════

□ 最大止損：任何持倉虧損不超過 7-8%（絕對上限）
□ 理想目標：大多數虧損控制在 5-6% 以內
□ 精確進場後的快速認錯：虧損 3-5% 即可止損
  （如果多個負面信號出現）
□ 止損點在進場前確定，寫下來
□ 止損一旦設定，只能往上移，絕不往下移
□ 止損被觸發時立即執行，沒有猶豫、沒有例外
□ 被止損出場後不立即追回同一隻股票
□ 偏好收盤價止損（而非盤中即時止損）
  → 但如果無法信任自己的紀律，使用自動止損

═══════════════════════════════════════════════════════
```

### 6.3 倉位管理規則

```
═══════════════════════════════════════════════════════
         📦 倉位管理規則
═══════════════════════════════════════════════════════

【核心公式】
倉位大小 = 單筆最大風險金額 ÷ 每股風險

其中：
• 單筆最大風險金額 = 帳戶總資金 × 1-2%
• 每股風險 = 買入價 - 止損價

【倉位限制】
□ 單筆風險：不超過帳戶的 1-2%
□ 集中持有：4-6 檔（最多 8-10 檔）
□ 最大單一倉位：不超過帳戶的 20-25%
□ 根據止損距離反向計算倉位大小
  → 止損越遠 → 倉位越小
  → 止損越近 → 倉位越大

【動態調整】
□ 虧損時縮小：連續虧損時減少倉位和頻率
□ 獲利時適度放大：帳戶增長後可逐步增加
□ 必須「贏得」交易更大倉位的權利

═══════════════════════════════════════════════════════
```

### 6.4 加碼規則——金字塔式加碼

```
✅ 正確做法（金字塔加碼）：
• 只對獲利的持倉加碼
• 每次加碼量遞減（底大頂小）
• 加碼後重新計算整體止損位

⛔ 禁止做法（倒金字塔/攤低成本）：
• 永遠不對虧損的持倉加碼
• 不「攤低成本」
```

### 6.5 回撤控制規則

```
┌──────────────┬──────────────────────────────────┐
│ 帳戶回撤幅度  │ 行動                             │
├──────────────┼──────────────────────────────────┤
│ < 3%         │ 🟢 正常交易                       │
│ 3-5%         │ 🟡 收緊止損，不開新倉             │
│ 5-7%         │ 🟠 減少持倉至 50%                 │
│ 7-10%        │ 🔴 只持有 1-2 個最強倉位           │
│ > 10%        │ ⛔ 全部清倉，停止交易，全面檢討    │
└──────────────┴──────────────────────────────────┘
```

### 6.6 連續虧損應對協議

```
┌──────────────┬──────────────────────────────────┐
│ 連虧次數      │ 行動                             │
├──────────────┼──────────────────────────────────┤
│ 連虧 2 筆    │ 下一筆倉位縮至正常的 75%          │
│ 連虧 3 筆    │ 倉位縮至正常的 50%                │
│ 連虧 4 筆    │ 暫停交易 2-3 天，重新評估         │
│ 連虧 5 筆    │ 暫停一週，全面檢討交易日誌         │
│ 回撤 >10%    │ 停止交易直到找到問題根源           │
└──────────────┴──────────────────────────────────┘

恢復路徑：小倉位試探 → 獲利確認 → 逐步增加 → 正常交易
```

### 6.7 風險回報比規則

```
□ 每筆交易風險回報比至少 2:1
□ 理想目標 3:1 或更高
□ 風險回報比不合格 → 直接放棄該交易，無論基本面多好

計算方式：
R:R = (目標價 - 買入價) ÷ (買入價 - 止損價)

例：買入 $50，止損 $47，目標 $62
R:R = ($62-$50) ÷ ($50-$47) = $12 ÷ $3 = 4:1 ✅
```

### 7.2 高潮頂部（Climax Top）信號清單

```
高潮頂部出現條件（需同時出現 3-4 個以上）：

□ 股價在最後 1-2 週加速上漲（比之前更陡）
□ 成交量暴增至平均量的 2-5 倍
□ 出現竭盡缺口（Exhaustion Gap）
□ 股價距離 50 日均線 >25-30%
□ 股價距離 200 日均線 >70-100%
□ 出現長上影線或高開低走
□ 主流媒體大量正面報導
□ 非交易者朋友開始問你這隻股票
□ 上漲已持續 18-20 週以上

→ 信號確認 → 賣出 50-75% 的倉位
```

### 7.3 追蹤止損方法

**方法一：利潤回撤法**

```
┌──────────────────┬──────────────────────────────┐
│ 帳面獲利幅度      │ 允許的最大回撤                │
├──────────────────┼──────────────────────────────┤
│ 獲利 5-10%       │ 移動止損至成本價（保本）       │
│ 獲利 10-15%      │ 允許從最高點回撤 5-7%         │
│ 獲利 15-20%      │ 允許從最高點回撤 7-10%        │
│ 獲利 20-30%+     │ 允許從最高點回撤 10-15%       │
└──────────────────┴──────────────────────────────┘
```

**方法二：均線追蹤法**

```
┌─────────────────┬─────────────────────────────────┐
│ 交易時間框架     │ 追蹤止損均線                     │
├─────────────────┼─────────────────────────────────┤
│ 短線（數日-數週）│ 10 日或 20 日均線               │
│ 中線（數週-數月）│ 50 日均線                       │
│ 長線（數月以上） │ 200 日均線                      │
└─────────────────┴─────────────────────────────────┘
收盤價跌破對應均線且不在 1-2 天內收回 → 賣出
```

**方法三：時間止損**

```
□ 買入後 2-3 週橫盤不動 → 考慮賣出
□ 買入後 4-5 週未獲利 5%+ → 強烈考慮賣出
□ 資金不應被困在「不動」的股票裡 → 機會成本
```

### 7.4 快速獲利規則

```
□ 1-3 週內獲利 15-20% → 賣出至少 50%
□ 例外：第一週就獲利 20%+ → 考慮持有至少 8 週
  （Power Play 信號）
```

---

### 7.5 永不讓獲利變虧損

```
核心規則：
□ 獲利達 5-10% → 必須將止損移至成本價
□ 此後帳面虧損的最壞情況 = 打平出場
□ 隨著獲利增加 → 持續上移止損
□ 止損只能上移，絕不下移
```

---

### 7.6 分批賣出策略

```
第一批（30-50%）：達到第一利潤目標 或 出現第一個弱勢信號
第二批（25-30%）：達到第二利潤目標 或 出現更多弱勢信號
第三批（剩餘）：追蹤止損被觸發 或 出現明確賣出信號
```

### 7.7 必須賣出的情境（無論你多看好）

```
□ 2.  跌破 50 日均線且 1-2 天內不收回
□ 3.  出現高潮頂部信號
□ 4.  獲利倉位回撤超過最大容許範圍
□ 5.  連續多天放量下跌（分銷信號）
□ 6.  盈餘低於預期且股價大跌
```

### 10.1 限制性信念 → 賦能性信念轉換表

```
┌─────────────────────────────┬─────────────────────────────┐
│ 限制性信念 🚫                │ 賦能性信念 ✅                 │
├─────────────────────────────┼─────────────────────────────┤
│ 虧損 = 我很笨               │ 虧損 = 經營成本               │
│ 好交易者每筆都賺             │ 好交易者管理風險長期盈虧正    │
│ 止損 = 認輸                  │ 止損 = 保護資本繼續戰鬥      │
│ 必須把虧的錢趕快賺回來       │ 專注下一筆高品質交易         │
│ 錯過大漲 = 巨大失敗          │ 市場總會提供新機會            │
│ 持有現金 = 浪費時間          │ 現金是彈藥，等待是策略       │
│ 賣了之後還漲 = 我是白痴      │ 按規則賣出 = 好的交易        │
│ 需要 100% 把握才能交易       │ 60% 把握 + 好的風險管理就夠  │
│ 這次不一樣不需止損           │ 每次都需要止損沒有例外       │
└─────────────────────────────┴─────────────────────────────┘
```

### 10.2 虧損處理八步法

```
1️⃣ 執行止損，立即退出
2️⃣ 深呼吸，離開螢幕 15 分鐘
3️⃣ 在交易日記中記錄
4️⃣ 問自己：是否遵守了規則？
5️⃣ 遵守了 → 「好的交易，不好的結果」
6️⃣ 沒遵守 → 找出原因並記錄
7️⃣ 不立即做下一筆交易
8️⃣ 回到觀察名單，評估下一個機會
```

---

### 10.3 「如果-那麼」情緒應對核心場景

```
如果：憤怒 / 想報復性交易
那麼：關閉軟體，離開 30 分鐘，今天不再交易

如果：恐懼 / 不敢進場
那麼：檢查設置是否符合所有規則，如果是 → 執行

如果：極度興奮 / 想加大倉位
那麼：堅持正常倉位，回顧最大虧損保持謙遜

如果：沮喪 / 想放棄
那麼：查看長期記錄，記住連虧是統計上必然的

如果：被洗出後不甘心 / 想追回
那麼：等待股票重新形成有效買入信號

如果：焦慮 / 無法安睡
那麼：你的倉位太大了——減倉到能安睡的水平
```

### 10.5 錯誤分類與改進優先級

```
🔴 A 類：紀律性錯誤（最高優先——投入 70% 改進精力）
   知道規則但沒遵守
   例：不止損、衝動交易、違反計劃、攤低成本

🟡 B 類：判斷性錯誤（中等優先——投入 25% 改進精力）
   遵守了規則但判斷有誤
   例：誤讀型態、基本面誤判、進場時機偏差

🟢 C 類：不可控錯誤（接受它——投入 5% 精力）
   做了一切正確的事但結果不好
   例：黑天鵝事件、意外壞消息
   → 這是交易的成本，風險管理會保護你
```
