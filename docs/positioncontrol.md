

# Martin Luk vs. Kristjan Kullamägi：倉位管理與出場策略比較

> **前言**：以下整理主要來自兩人在公開訪談（Podcast、社交媒體、講座）中分享的方法論。Kristjan 因長期在 Twitter/X 和影片中大量公開其系統細節，資料較豐富；Martin 相對較少公開完整細節，部分內容是根據其訪談與社群分享所歸納。

---

## 一、核心理念差異總覽

| 面向 | Martin Luk（香港） | Kristjan Kullamägi（瑞典） |
|---|---|---|
| **持倉集中度** | 高度集中（通常同時 3–6 檔） | 較分散（可同時 8–20+ 檔） |
| **單筆風險** | 相對較大（可達帳戶 1–2%+） | 較小且嚴格（通常 0.25–0.75%） |
| **加碼方式** | 傾向一次性或少次大手建倉 | 漸進式金字塔加碼（多次小批加倉） |
| **持倉時間** | 偏中長線波段（數週至數月） | 偏短中線波段（數天至數週為主） |
| **出場哲學** | 讓贏家盡量跑，容忍較大回撤 | 系統化分批止盈 + 移動止損 |
| **止損寬度** | 較寬（基於週線/日線結構） | 較窄（基於日線精確技術位） |

## 二、倉位大小（Position Sizing）的計算

### 兩人共通的基礎公式

$$\text{股數} = \frac{\text{每筆願意承受的虧損金額（$R）}}{\text{買入價} - \text{止損價}}$$

這個公式兩人都使用，但參數設定差異很大：

---

### Kristjan 的做法：小 R、多筆、金字塔

**核心參數：**
- 帳戶風險 $R$ = 帳戶總值的 **0.25% ~ 0.75%**（A 級設定可能到 1%）
- 初始只建 **1/3 ~ 1/2 的目標倉位**
- 股票證明自己後（突破、放量），再加碼至全倉甚至超額

**例子：**
> 帳戶 $1,000,000
> - 單筆風險 = 0.5% = **$5,000**
> - 某股準備在 $50 買入，止損設在 $47（風險 $3/股）
> - 可買股數 = $5,000 ÷ $3 = **1,667 股**
> - 初始先建 **800 股**（約半倉）
> - 股票突破新高後，再加 **867 股**，同時將止損上移

**Portfolio Heat（總曝險）概念：**
Kristjan 會監控所有持倉的合計風險，通常控制在帳戶的 **3–6%** 以內。例如同時持有 12 個倉位，每個風險 0.5%，總風險 = 6%。

---

### Martin 的做法：大注、集中、一步到位

**核心參數：**
- 單筆風險可達帳戶的 **1–2%** 甚至更多（高確信設定時）
- 傾向一次性建立 **全倉或接近全倉**
- 持倉數少，因此單一持倉佔帳戶的比重大（可達 15–30%+）

**例子：**
> 帳戶 $1,000,000
> - 單筆風險 = 1.5% = **$15,000**
> - 某股在 $100 買入，止損設在 $94（風險 $6/股）
> - 可買股數 = $15,000 ÷ $6 = **2,500 股**
> - 倉位價值 = $250,000（佔帳戶 25%）
> - **一次性建倉**，除非盤面流動性不夠才分批

**重點差異：** Martin 能承受較大的日內波動（因為止損較遠），但總持倉檔數少，所以總帳戶風險未必比 Kristjan 高太多。

## 三、進場後的出場方法（重點部分）

### A. Kristjan 的出場系統：「分批 + 移動均線尾隨」

Kristjan 的出場是高度系統化的，可以拆解為以下幾個部分：

#### ① 硬止損（Initial Stop Loss）

- 設在技術位下方，例如：
  - 突破形態的**底部**
  - 關鍵支撐位下方
  - 或買入價下方 **1–1.5 個 ATR**
- 止損一旦設定，**絕不往下移**

```
止損例子：
買入 $50，形態底部在 $47.50
→ 止損設 $47.20（給一點緩衝）
→ 單股風險 = $2.80
```

#### ② 時間止損（Time Stop）

- 如果買入後 **3–5 個交易日**，股票沒有明顯往有利方向移動 → **出場**
- 邏輯：好的突破通常會很快發動，如果「死」在那裡，說明市場不認同

```
例子：
週一買入 $50，到週五收盤仍在 $50–51 之間橫盤
→ 週五尾盤或下週一開盤出場，不等止損被觸發
```

#### ③ 分批止盈（Scaling Out Into Strength）

- 股票上漲後，分批鎖定利潤：
  - 第一批（約 1/3）：在 **1R–2R**（賺到風險金額的 1–2 倍）處賣出
  - 第二批（約 1/3）：在 **3R–5R** 或到達明顯阻力位時賣出
  - 最後一批（1/3）：用移動均線尾隨，盡量「坐大浪」

```
例子（R = $2.80/股，買入 1,500 股 @ $50）：

第一批 500 股：在 $55.60 賣出（= $50 + 2×$2.80），鎖定 $2,800
第二批 500 股：在 $64.00 賣出（= $50 + 5×$2.80），鎖定 $7,000
第三批 500 股：用 10 EMA 尾隨，股票最終在 $72 跌破 10 EMA → 出場
                鎖定 $11,000

總利潤 = $20,800，平均約 5R
```

#### ④ 移動均線尾隨止損（Trailing MA Stop）

Kristjan 最常用的尾隨方法：

| 階段 | 尾隨均線 | 情境 |
|---|---|---|
| 初期突破、快速上漲 | **10 EMA**（日線） | 股票陡峭上漲，動能強 |
| 中期趨勢 | **20 EMA**（日線） | 上漲放緩但趨勢仍在 |
| 大波段行情 | **50 SMA**（日線） | 股票已持有數週，想坐更長 |

**規則：** 收盤價跌破該均線 → 次日出場（不是盤中刺破就跑）

```
例子：
持有股票從 $50 漲到 $75，一路用 10 EMA 尾隨
$75 後上漲速度放慢，改用 20 EMA 尾隨
當 20 EMA 在 $68，某天收盤 $67.50（跌破）
→ 次日開盤出場
```

#### ⑤ 高潮頂/拋物線出��（Climax Top）

- 如果股票出現**拋物線加速**（parabolic move）：
  - 一天漲 15–30%+
  - 成交量暴增
  - 長上影線
- → Kristjan 會主動**賣出大部分或全部**，不等均線止損

---

### B. Martin 的出場系統：「讓贏家跑 + 結構性止損」

Martin 的出場邏輯更偏向「儘量不被洗出去」，願意忍受更大的回撤來換取更大的波段利潤。

#### ① 初始止損（較寬）

- 設在**週線級別**的關鍵結構位下方
- 或日線上明顯的波段低點下方
- 止損空間通常比 Kristjan 大

```
例子：
買入 $100，最近的週線支撐在 $90
→ 止損設在 $89（風險 $11/股，約 11%）

對比 Kristjan 可能在同樣的股票上：
買入 $100，日線突破形態底部 $96
→ 止損設 $95.50（風險 $4.50/股，約 4.5%）
```

**關鍵差別：** Martin 的止損寬了一倍多，但因此他的倉位（股數）會相對少。

#### ② 較少分批賣出，傾向全進全出

- Martin 傾向持有完整倉位，直到**趨勢出現明確結構性破壞**
- 不太做「賺 2R 先賣 1/3」這種操作
- 他的邏輯：過早獲利了結會砍掉大贏家的潛力

```
例子：
買入 $100，股票漲到 $150
Kristjan 此時可能已賣掉 2/3 的倉位
Martin 可能仍持有 100% 的倉位，止損上移到 $130 附近
```

#### ③ 結構性止損上移（Structural Trailing Stop）

- 不依賴固定均線，而是隨著股票形成**新的支撐結構**來上移止損
- 例如：股票每形成一個新的盤整區間（base），止損就移到該區間底部

```
圖示化概念：

買入 $100，初始止損 $89
↓
股票漲到 $120，回調形成支撐在 $112
→ 止損上移到 $111
↓  
股票漲到 $145，回調形成支撐在 $135
→ 止損上移到 $134
↓
股票漲到 $180，某天跌破 $160 的支撐結構
→ 出場
```

#### ④ 長期移動均線參考

- 當持倉時間較長時，Martin 可能參考 **50 日均線** 或 **10 週均線**（本質上類似）
- 但不是機械式「收盤跌破就跑」，而是結合成交量和價格行為判斷

#### ⑤ 基本面/故事惡化時出場

- Martin 也關注股票的「故事」是否改變
- 如果公司的增長邏輯被打破（例如財報大幅不及預期、行業政策變化），即使技術面止損未觸發，也可能主動出場

## 四、具體對比情境

### 假設同一檔股票，兩人同時進場：

**股票 XYZ：$50 突破橫盤區間，放量上漲**

| | Kristjan | Martin |
|---|---|---|
| **買入價** | $50 | $50 |
| **初始止損** | $47.50（形態底部） | $44（週線支撐） |
| **單股風險** | $2.50 | $6.00 |
| **帳戶風險** | 0.5% = $5,000 | 1.5% = $15,000 |
| **股數** | 2,000 股 | 2,500 股 |
| **倉位金額** | $100,000（10%） | $125,000（12.5%） |

**股票走到 $65 時（+30%）：**

| | Kristjan | Martin |
|---|---|---|
| **動作** | 已賣 700 股 @ $55（1R），再賣 700 股 @ $62.50（5R）；剩 600 股用 10EMA 尾隨 | 仍持有全部 2,500 股，止損上移到 $58（新支撐結構） |
| **已實現利潤** | $3,500 + $8,750 = $12,250 | $0（未實現 $37,500） |
| **剩餘曝險** | 600 股，均線尾隨 | 2,500 股，結構止損在 $58 |

**股票漲到 $80 後回落到 $68：**

| | Kristjan | Martin |
|---|---|---|
| **結果** | 10EMA 約在 $72 → 跌破出場 600 股 @ ~$71；總利潤 ≈ $12,250 + $12,600 = **$24,850** | 結構支撐可能在 $70 → 跌破出場 2,500 股 @ ~$69；總利潤 = 2,500 × $19 = **$47,500** |

**但如果股票從 $65 直接跌回 $50：**

| | Kristjan | Martin |
|---|---|---|
| **結果** | 已實現 $12,250；剩 600 股止損出場 @ ~$58（10EMA）；剩餘利潤 600 × $8 = $4,800；**總利潤 ≈ $17,050** | 止損 $58 觸發，2,500 股 @ $58；總利潤 = 2,500 × $8 = **$20,000** |

**如果股票直接跌破止損（假設跳空到 $43）：**

| | Kristjan | Martin |
|---|---|---|
| **虧損** | 2,000 × ($50–$43) = **–$14,000**（1.4%帳戶） | 2,500 × ($50–$43) = **–$17,500**（1.75%帳戶） |

→ 兩人虧損都超出預期，但因 Kristjan 初始倉位相對較小、風險預算低，受損略少。

## 五、關鍵差異總結

```
                  Kristjan                          Martin
               ┌──────────────┐              ┌──────────────┐
 風險控制      │  小注分散     │              │  大注集中     │
               │  嚴格 R 控制  │              │  更大容錯空間 │
               └──────┬───────┘              └──────┬───────┘
                      │                              │
 加碼          │  金字塔逐步加 │              │  一次到位     │
               └──────┬───────┘              └──────┬───────┘
                      │                              │
 出場          │  分批止盈     │              │  全倉持有     │
               │  均線尾隨     │              │  結構性止損   │
               │  時間止損     │              │  趨勢破壞才走 │
               └──────┬───────┘              └──────┬───────┘
                      │                              │
 結果特徵      │  勝率較高     │              │  單筆利潤更大 │
               │  波動較小     │              │  帳戶波動更大 │
               │  穩定複利     │              │  爆發式增長   │
               └──────────────┘              └──────────────┘
```

### 用一句話總結：

- **Kristjan**：像一個**精密的系統**——用很多小額賭注、嚴格止損、分批收割，穩穩地把期望值磨出來。
- **Martin**：像一個**重拳出擊的狙擊手**——看準機會就大筆押注、用寬止損忍受波動、讓少數大贏家驅動帳戶增長。
