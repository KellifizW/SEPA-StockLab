## 第十三章：獲利交易深度解剖

### 13.1 Winners 的核心法則

🧠 **心智模型：大贏家來自在正確的時間買入，而非隨機買入**

> 「如果你在正確的時間買入，大多數大贏家真的來自在正確的時間買入，而不是非常隨機的那些。這就是為什麼研究你過去的交易很重要——你可以找到模式或從中獲得啟發。」

**Winners vs Losers 的關鍵區別：**

| 維度 | Winners | Losers |
|------|---------|--------|
| 進場時機 | 在支撐匯聚區精準進場 | 在隨機位置或阻力位進場 |
| EMA 方向 | 上升/趨勢明確 | 下降/模糊/混亂 |
| 日線結構 | Higher Lows、Base 內第一次回調 | 沒有 Higher Low、結構鬆散 |
| Weekly 確認 | 趨勢健康 | 被忽略或矛盾 |
| 心理狀態 | 冷靜、系統化 | FOMO、急躁、報復交易 |

### 13.2 BBAI——Breakout 結合主題識別

**交易概覽：**

| 項目 | 詳情 |
|------|------|
| 類型 | Breakout（Gap Up Day） |
| 時間 | 2025年2月初 |
| 主題 | AI 概念股（當時仍是熱門主題） |
| 進場 | 1分鐘 Opening Range High Breakout |
| 止損 | 較寬（2025年初期尚未完全收緊止損） |

**Martin 喜歡這個 Base 的原因：**
1. **持續建立 Higher Lows**——買方逐步掌控
2. **強勢 Reclaim**——跌破後快速收復
3. 填補了之前的未填補缺口（Unfilled Gap）

**引起 Martin 注意的關鍵日：**

```
Day 1：星期四 — Gap Up + 強勢收盤 + 收復了之前的缺口
       → 「顯示力量」（Showing Strength）
       → 不像弱勢股那樣在 EMA 被拒絕後繼續下跌

Day 2：星期一 — 開盤走弱 + 填補 Day 1 的缺口 + 找到支撐
       → 強勢反轉，收盤高於所有 EMA
       → 這可能是另一個 Higher Low
       
Day 3：Gap Up 突破 → 進場！
```

⚠️ **Martin 承認的不足：**
- 他希望 Base 能多盤整 1-2 天再突破，這樣會更緊
- 但2025年的市場特徵是股票不會給你那麼有序的盤整——「它們就是直接走」

**賣出執行：**
- 每次覺得 extended → 賣出 ~15% 倉位
- Trail 剩餘倉位
- 最終在**第一次收盤跌破 9 EMA 時**平倉全部

🔴 **硬規則：每次部分獲利了結 = ~15% 的倉位**

> 「我通常每次覺得這個股票 extended 的時候就賣 15%。」

🧠 **心智模型：認識自己的弱點並優先修復**

1. **最高優先**：控制下行（減少過度交易、避免隨機交易）

### 13.8 PL（Planet Lab）——EP Pullback（Opening Range 假突破後的進場）

**交易概覽：**

| 項目 | 詳情 |
|------|------|
| 類型 | EP（Episodic Pivot）中的 Pullback |
| 2025年典型模式 | 開盤突破 → 立即回撤 → 找到支撐 → 真正的突破 |
| 進場 | 5分鐘 Candle Breakout（回撤後的反彈） |
| 止損 | 5分鐘 Breakout Candle Low → 2.3%（< 2.5% ✅） |

**這是2025年市場的典型特徵：**
> 「2025年有很多開盤時的假突破。Opening Range Highs 經常是 Fakeout——它突破但又回來，然後我買5分鐘 Candle Breakout。」

**賣出與教訓：**
- 全部賣入強勢 → 之後又漲了 30%
- 如果用 9 EMA trailing：可能再捕捉 10%
- Martin 認為這是可以接受的——至少捕捉了上漲趨勢的一大部分

**Position Sizing 示範：**

Martin 在這筆交易中詳細展示了他的即時計算過程：

```
Step 1: 5分鐘 candle 形成後，計算 breakout 到 candle low 的距離
        → 假設 = 2.5%

Step 2: 計算 breakout 到 Low of Day 的距離  
        → 假設 = 3%（太寬 → 選擇 candle low 作為止損）

Step 3: 確定帳戶風險
        → 假設帳戶 $10,000，風險 0.5% = $50

Step 4: 計算倉位
        → $50 ÷ 2.5% = $2,000 position
        → $2,000 ÷ $10,000 = 20% position size

Step 5: 下單
```

📊 **數學邏輯：快速心算的經驗法則**

> 「當你經常做的時候，你會知道 2% 的止損 = 大約 25% 的倉位。你有一個很快的心算。而且你的倉位通常不會在連續交易之間劇烈變化——通常在 20-30% 之間。」

### 13.9 IONQ——從 Weekly 到 Intraday 的完美執行

**交易概覽：**

| 項目 | 詳情 |
|------|------|
| 類型 | Breakout（Quantum 主題） |
| 時間 | 2025年9月初 |
| 行業 | 量子計算（Quantum Computing） |
| R-Multiple | ~20-30R |

🧠 **心智模型：Weekly Chart 可以讓 Daily 混亂的圖表變得清晰**

> 「在日線上看起來有點亂——橫盤震盪、跌破 50、回調到 150。但在週線上，它看起來乾淨得多——特別是這根巨大的 Pin Bar 和 Inside Week。這兩根週線柱簡直在對我大喊：'這個設置好了，它可能是另一個領導股。'」

**IONQ Weekly 的關鍵特徵：**
1. **Pin Bar**（長下影線）→ 強烈的買方參與
2. **Inside Week**（隨後的一週完全在 Pin Bar 範圍內）→ 波動收縮
3. 組合起來 = **Weekly VCP / Cup with Handle** 的特徵

**執行流程：**
1. 在 Weekly 上發現機會
2. 之前嘗試過幾次進場都被止損（Daily chart 上的前期嘗試）
3. 但 Weekly 仍然看起來很好（上方 50 EMA，consolidating tight）
4. 再次嘗試 → 這次在 Breakout Day 有效
5. 5分鐘 Candle Breakout 或 Opening Range High

**關鍵教訓——一筆贏家如何補償多筆虧損：**

📊 **數學邏輯：Tight Stop 的最強驗證**

> 「你可以看到一筆好的交易可以給你超過 20R 甚至 30R。所以你可以補償你之前 30 次的虧損交易。如果你在極端情況下被止損了 30 次然後才進入這筆交易，你仍然可以盈虧打平。」

| 場景 | 計算 |
|------|------|
| 止損 = 0.5% 帳戶 × 30 次 | = -15% 帳戶 |
| 一筆 30R winner at 0.5% risk | = +15% 帳戶 |
| 淨結果 | = 盈虧平衡 |

**賣出：**
- 賣了更多到強勢（比 BBAI 更積極）
- 最後 ~80% 賣入弱勢
- Martin 承認他又一次因為「覺得市場 extended」（9月）而提前賣出

## 結語

Martin Luk 的交易系統之所以能產生 +969% 的回報，不是因為某一個神奇的指標或秘密的設置，而是因為：

1. **極致的風控紀律**：每筆交易只承受 0.5% 帳戶風險
2. **數學優勢**：Tight Stop 的拋物線式回報提升

5. **持續學習和自我批評**：即使在 +969% 的年份，仍然詳細分析每一筆虧損
6. **接受不完美**：22% 的勝率、錯過的領導股、12月的26%回撤——全部被接受為系統的一部分

**最重要的是**：Martin 經歷了2021年的50%回撤、2022年的幾乎零回報、2.5年才收復失地——然後才開始了連續兩年的三位數甚至四位數回報。這個旅程本身就是最大的教訓。
