# Section 1：策略核心定義與哲學

## 1.1 一句話定義

> **「我們要尋找強勢、波動大的動量股，在它們從整理形態中突破的那一刻買入。」**
>
> *"We are going to look for strong, volatile momentum stocks that are breaking out of a consolidation pattern."*

## 1.2 策略的五根支柱

```
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│ 支柱①    │ 支柱②    │ 支柱③    │ 支柱④    │ 支柱⑤    │
│ 動量領導  │ 高波動性  │ 整理形態  │ 均線支撐  │ 放量突破  │
│          │          │          │          │          │
│ 過去1-6月 │ ADR ≥ 5% │ Higher   │ 騎乘/彈跳 │ 衝破區間  │
│ 翻倍以上  │ 理想 ≥8% │ Lows +   │ 20SMA    │ 上緣 +   │
│ 買高賣更高│ 快速爆發  │ 收緊     │ 均線向上  │ 成交量放大│
└──────────┴──────────┴──────────┴──────────┴──────────┘
```

## 1.3 核心哲學

```
┌──────────────────────────────────────────────────────────────┐
│                                                               │
│   ① 買高賣更高（Buy High, Sell Higher）                       │
│   → 我們不買便宜的股票，我們買最強的股票                       │
│   → 趨勢的延續比趨勢的反轉更常見                              │
│                                                               │
│   ② 圖表 = 供需地圖                                           │
│   → 我們的工作是正確閱讀供需，在資金流入的方向交易              │
│   → Higher Lows = 需求持續增強的物理證據                       │
│   → 收緊 = 供需即將失衡，突破即將發生                          │
│                                                               │
│   ③ 股票以階梯方式運動                                         │
│   → 漲 → 整理 → 漲 → 整理 → 漲 → 整理                       │
│   → 我們的工作是在每一級台階開始形成的那一刻買入                │
│                                                               │
│   ④ 純技術交易                                                 │
│   → 不使用基本面分析（交易時間太短，基本面無影響）              │
│   → 所有決策基於圖表、均線、成交量和價格行為                   │
│                                                               │
│   ⑤ 流程導向                                                   │
│   → 職業交易者關注改進流程，而不是追逐下一筆交易                │
│   → 擁有可重複、可改進的系統                                   │
│   → 每天掃描、每筆記錄、每週復盤、每月評估                     │
│                                                               │
│   ⑥ 錯的時候少虧，對的時候多賺                                 │
│   → 勝率可能只有 30-40%                                       │
│   → 但平均獲利遠大於平均虧損                                   │
│   → 長期期望值為正                                             │
│                                                               │
│   ⑦ 小帳戶的不公平優勢                                         │
│   → 小帳戶可以交易大基金無法交易的高波動小型股                  │
│   → 這是你唯一的結構性優勢                                     │
│   → 如果你用小帳戶交易慢速大型股，你自願放棄了這個優勢          │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```

## 2.2 核心行為原則

```
╔══════════════════════════════════════════════════════════════════╗
║                                                                  ║
║   原則 A：閱讀供需地圖                                           ║
║   → 圖表是供需的記錄，不是預測工具                              ║
║   → Higher Lows = 機構累積的腳印                                ║
║   → 我們的工作 = 在資金流入的方向交易                           ║
║                                                                  ║
║   原則 B：必須「早」進場                                         ║
║   → 突破瞬間就行動，不能晚 2 小時才追                           ║
║   → 早進場 = 止損更近 + 回報更大 = 風險回報比極好               ║
║   → 晚進場 = 止損更遠 + 回報更小 = 風險回報比很差               ║
║   → 前一晚做好分析 → 警報一響 → 確認 → 立刻執行                ║
║                                                                  ║
║   原則 C：假突破不代表 Setup 壞了                                ║
║   → 假突破後如果 Higher Lows 仍在 + 區間更緊 → setup 更強       ║
║   → 弱手被洗出去 → 供給更少 → 下次突破更爆發                   ║
║   → 「The stock doesn't care that you got frustrated with it」  ║
║   → 用結構判斷，不用情緒判斷                                    ║
║                                                                  ║
║   原則 D：乾淨 vs 雜亂                                           ║
║   → 好的圖表：均線整齊向上、價格有秩序、Higher Lows 清晰        ║
║   → 壞的圖表：均線糾纏、價格亂跳、看了頭痛                     ║
║   → 頭痛的圖表 → 2 秒內關掉 → 絕對不碰                        ║
║                                                                  ║
║   原則 E：只交易動量領導者                                       ║
║   → 不是「所有上漲的股票」而是「漲得最多最快的那些」             ║
║   → 「碰巧在均線上找到支撐」的隨機股 ≠ 好 setup                ║
║   → 不是領導者 → 不碰                                          ║
║                                                                  ║
║   原則 F：耐心是王                                               ║
║   → 等待 setup 成熟（20SMA 追上來 + 區間收緊到極致）            ║
║   → 不要急著進場                                                ║
║   → 寧可錯過一筆交易，也不要在不夠好的 setup 上浪費資金         ║
║                                                                  ║
║   原則 G：研究數千張歷史圖表                                     ║
║   → 股票今天的走勢和 100 年前一樣                               ║
║   → 模式反覆出現                                                ║
║   → 你看過越多圖表，辨識模式的能力越強                          ║
║   → 建立圖表資料庫 → 截圖保存每一個好 setup                    ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```

# Section 5：五大選股標準 — 完整判斷規則

## 5.1 標準之間的關係：篩選漏斗

```
整個市場（數千支股票）
    ↓ 標準① 動量：過去 1-6 個月大幅上漲 → 篩掉 ~90%
    ↓ 標準② ADR：≥ 5% → 再篩掉一大批慢股
    ↓ 標準③ 整理形態：Pennant/Flag/Higher Lows → 只留正在蓄力的
    ↓ 標準④ 均線支撐：騎乘/彈跳 20SMA → 確認趨勢健康
    ↓ 標準⑤ 放量突破：價格衝破區間 + 放量 → 你的進場信號
    ↓
📌 你的 Watchlist（通常 5-15 支）

標準 ①② = Scanner 自動完成（數字篩選）
標準 ③④ = 你的眼睛和經驗完成（視覺判斷）
標準 ⑤  = 市場告訴你（等待突破發生）
```

## 5.2 標準①：強動量（Momentum Leadership）

```
╔══════════════════════════════════════════════════════════════════╗
║                                                                  ║
║   定義：過去 1-6 個月中漲幅最大的那一批股票                       ║
║                                                                  ║
║   Scanner 設定：                                                  ║
║   • 1個月 ≥ 25% 漲幅                                            ║
║   • 3個月 ≥ 50% 漲幅                                            ║
║   • 6個月 ≥ 150% 漲幅                                           ║
║                                                                  ║
║   判斷標準：                                                      ║
║   ✅ 是板塊中最強的那批                                           ║
║   ✅ 有相對強勢（大盤跌時它少跌或不跌）                           ║
║   ✅ 有機構資金累積的證據（Higher Lows）                          ║
║   ✅ 有「自然水流」（不需要外力推動就會漲）                       ║
║                                                                  ║
║   排除標準：                                                      ║
║   ❌ 「碰巧看起來不錯」的隨機股                                   ║
║   ❌ 純散戶推動、沒有自然水流的微型股                             ║
║   ❌ 已經漲了很多但回調幅度太大（動量可能消散）                   ║
║                                                                  ║
║   ⚠️ 反直覺要點：                                                ║
║   → 「已經漲了 300%」不代表「太貴了」                            ║
║   → 只要趨勢健康（均線向上 + Higher Lows）                      ║
║   → 它可以繼續漲更多                                            ║
║   → 趨勢的延續比趨勢的反轉更常見                                ║
║                                                                  ║
║   相對強勢（Relative Strength）：                                 ║
║   → 不是 RSI 指標                                               ║
║   → 而是一支股票相對於大盤或同類股票的表現                       ║
║   → 大盤跌 -5%，Stock A 只跌 -1% → Stock A 有相對強勢           ║
║   → 相對強勢 = 有更強的買盤支撐                                 ║
║   → Qullamaggie 甚至會因此打破其他規則                           ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```

## 5.3 標準②：高 ADR（Average Daily Range）

```
╔══════════════════════════════════════════════════════════════════╗
║                                                                  ║
║   定義：過去 14 天，每天的（最高價/最低價 - 1）的平均值           ║
║   → 告訴你這支股票每天「能動多少」                               ║
║                                                                  ║
║   ADR 門檻規則：                                                  ║
║   ─────────────                                                 ║
║   < 3%      ❌❌ 絕對不碰（「shouldn't be on your scans」）       ║
║   3% - 5%   ❌  太慢（即使 setup 是 5 星也 pass）                ║
║   5% - 8%   ✅  可接受的最低門檻                                 ║
║   8% - 15%  ✅✅ 理想範圍                                        ║
║   15%+      ✅✅✅ 極度爆發（但注意流動性）                       ║
║                                                                  ║
║   ⚡ 核心規則：                                                   ║
║   → 帳戶 < $1M → 不看 ADR < 5% 的股票                          ║
║   → ADR 是你「第一個看的指標」                                   ║
║   → ADR 有「獨立否決權」：                                      ║
║     即使 setup 的其他維度全部完美（= 5 星），                    ║
║     如果 ADR 太低，仍然不交易                                   ║
║                                                                  ║
║   帳戶大小與 ADR 的關係：                                         ║
║   < $100K     偏好 ADR ≥ 8%                                     ║
║   $100K-$500K  ADR ≥ 5%                                         ║
║   $500K-$1M    ADR ≥ 4-5%                                       ║
║   > $1M        ADR ≥ 3-4%（流動性限制）                         ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```

## 5.4 標準③：整理形態（Consolidation Pattern）

```
╔══════════════════════════════════════════════════════════════════╗
║                                                                  ║
║   定義：大漲後橫盤交易，價格在逐漸收窄的區間內波動                ║
║   理想形態：Pennant（三角旗形）                                   ║
║                                                                  ║
║   Qullamaggie 實際上在看什麼（按重要性排序）：                    ║
║                                                                  ║
║   1️⃣ Higher Lows（越來越高的低點）— 最重要                       ║
║      → 直播中出現頻率最高的概念                                  ║
║      → 每一個更高的低點 = 新買家在更高價位進場                   ║
║      → 需求正在增強                                             ║
║      → 「Relentlessly building higher lows」                     ║
║      → 「Building higher lows means it's getting accumulated」  ║
║                                                                  ║
║   2️⃣ Tightness（收緊程度）                                      ║
║      → K 線越來越小，區間從寬變窄                               ║
║      → 收緊 = 供需接近完美平衡 = 突破時爆發力更強               ║
║      → 像彈簧：壓得越緊，釋放時彈得越高                         ║
║      → 「Wow this thing is so tight」                            ║
║                                                                  ║
║   3️⃣ 清晰的區間（Clean Range）                                   ║
║      → 你能用兩條線明確畫出整理區間的上緣和下緣                  ║
║      → 區間模糊 = setup 不乾淨 = pass                           ║
║                                                                  ║
║   4️⃣ 整理時間                                                    ║
║      → 太短：還沒充分消化前一波漲幅                              ║
║      → 太長：動量可能消散                                       ║
║        「Been going sideways too long」→ pass                    ║
║      → 理想：3-15 天（高動量名稱可以只有 3-5 天）               ║
║                                                                  ║
║   成交量模式：                                                    ║
║   → 大漲期間：放量                                              ║
║   → 整理期間：縮量（沒有人急著賣 = 供給稀缺 = 正常且理想）      ║
║   → 突破時：再次放量                                            ║
║                                                                  ║
║   Scanner 如何篩選整理：                                          ║
║   → Price within 15% of 6-day high + Price within 15% of        ║
║     6-day low = 過去 6 天在窄區間內交易                          ║
║   → 真正的判斷仍然需要你用眼睛看圖表                            ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```

## 5.5 標準④：均線支撐（Moving Average Alignment）

```
╔══════════════════════════════════════════════════════════════════╗
║                                                                  ║
║   三條關鍵均線：                                                  ║
║                                                                  ║
║   10 SMA 🏎️「快速車道」                                          ║
║   → 最強勢股票在初始爆發階段騎乘 10SMA                          ║
║   → 在 10SMA 上的突破有更高失敗率                               ║
║   → 主要用作獲利後的 trailing stop                              ║
║                                                                  ║
║   20 SMA ⭐「黃金線」— 最重要的均線                               ║
║   → 整個系統的核心參考                                          ║
║   → 大多數最好的 setup 在 20SMA 附近整理                        ║
║   → 「There is something truly magical about that yellow         ║
║      line. I don't know what or why it is, but it works.」       ║
║   → 小帳戶應該優先找 20SMA setup                                ║
║                                                                  ║
║   50 SMA 🐢「慢速支撐」                                          ║
║   → 較慢的股票在 50SMA 上整理                                   ║
║   → 也可以交易，但較慢                                          ║
║   → 小帳戶優先找 20SMA setup                                    ║
║                                                                  ║
║   其他均線（特殊情況）：                                          ║
║   → 65 EMA（60分鐘圖）、100/150/200 SMA（日線）                ║
║   → 20 SMA（週線）                                              ║
║   → 主要用於反向拋物線做多等進階 setup                          ║
║                                                                  ║
║   ──────────────────────────────────────────────                ║
║                                                                  ║
║   均線方向比位置更重要：                                          ║
║   ✅ 向上傾斜（Rising）→ 趨勢健康                               ║
║   ❌ 向下傾斜（Sloping Down）→ 趨勢惡化                         ║
║   ❌ 糾纏在一起（All Over The Place）→ 沒有趨勢 = 不碰          ║
║                                                                  ║
║   「Surfing」的完整定義：                                         ║
║   ① 均線持續向上傾斜                                            ║
║   ② 股價大部分時間在均線上方                                    ║
║   ③ 偶爾回調觸碰均線，被托住                                    ║
║   ④ 形成 Higher Lows，低點在均線附近                             ║
║   ⑤ 然後再次上漲                                                ║
║                                                                  ║
║   均線的階梯式「層級降級」：                                      ║
║   最強勢 → Surfing 10SMA                                        ║
║   健康趨勢 → Surfing 20SMA（最常見的好 setup 位置）             ║
║   較慢趨勢 → Surfing 50SMA                                      ║
║   危險 → 跌破 50SMA（趨勢可能結束）                             ║
║                                                                  ║
║   Undercut & Reclaim（跌破後快速收回）：                          ║
║   → 短暫跌破均線後快速收回 = 仍然健康，甚至是買入機會            ║
║   → 但持續在均線下方 = 趨勢轉弱                                 ║
║                                                                  ║
║   等待「20SMA 追上來」：                                          ║
║   → 大漲後橫盤，股價在高位但 20SMA 還在下面                     ║
║   → 如果繼續橫盤，20SMA 逐日上升                                ║
║   → 幾天後 20SMA 追上股價 → 這時突破 = 最高品質                 ║
║   → 股價遠離 20SMA 的突破失敗率更高                             ║
║   → 「Patience is king」                                        ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```

## 5.7 五大標準完整決策樹

```
Scanner 輸出一支股票 → 開始逐一檢查：

① 動量：過去 1-6 個月漲了多少？
   → 不符合 → ❌ PASS
   → 符合 → 繼續 ↓

② ADR：具體數字是多少？
   → < 5% → ❌ PASS（「shouldn't be on your scans」）
   → ≥ 5% → 繼續 ↓

③ 整理形態：看日線圖
   → 沒有整理/正在暴漲/正在暴跌 → ❌ PASS
   → 雜亂（choppy）→ ❌ PASS（「headache」）
   → 乾淨整理 + Higher Lows + 收緊 → ✅ 繼續 ↓

④ 均線：股價和均線的關係？
   → 均線全部向下或糾纏 → ❌ PASS（「not surfing」）
   → 股價遠離 20SMA → ⏳ 等待（「wait for 20 day to catch up」）
   → 騎乘/彈跳 50SMA → ⚠️ 可以但較慢
   → 騎乘/彈跳 20SMA + 均線全部向上 → ✅ 加入 Watchlist ↓

⑤ 突破：有沒有突破？
   → 還沒 → 📋 Watchlist + 設警報 → 等待
   → 突破但量不大/價不快 → ⚠️ 觀望（可能假突破）
   → 放量突破 + 價格快速上升 → 🎯 進場！
```

# Section 7：進場規則 — 完整決策流程

## 7.1 日內突破進場（最常見情境）

```
IF   股價突破你畫的整理區間上緣
AND  成交量放大
AND  價格快速上升
THEN → 立刻買入
     → 不猶豫，不多想（前提是前一晚已做好分析）
     → 「As soon as it starts breaking, I buy. I don't second guess. I don't think.」

止損：買入後第一個動作 = 設止損單在當日低點下方
倉位：根據星級（見 Section 6.2）
```

## 7.2 盤前跳空進場

```
跳空幅度判斷：
→ > 10% → 通常 PASS
→ ≤ 10% → 繼續評估

IF 跳空 ≤ 10%：
→ 不要在開盤瞬間買入！
→ 等 1-5 分鐘觀察開盤後價格行為
→ IF 價格停滯/下跌 → 不買
→ IF 放量繼續上漲 → 買入

Opening Range High（ORH）技巧：
→ 等第一根 K 線（1分鐘/5分鐘/60分鐘）形成
→ 高點 = ORH
→ 後續突破 ORH → 買入信號（確認跳空有後續力量）
→ 跌破 ORL → 不買（跳空可能失敗）
→ 1分鐘 ORH = 最激進 | 5分鐘 = 平衡 | 60分鐘 = 最保守
```

## 7.3 「早」的具體含義

```
✅ 「早」= 在突破開始的那一刻就買入
   → 股價剛碰到或穿過區間上緣
   → 你在前一晚已經做好分析
   → 警報一響 → 確認放量 → 立刻執行

❌ 「早」不是：
   → 不是在突破前就買入（那叫 anticipating）
   → 不是盲目追高（必須有預先分析）
   → 不是每次都不加思考（有些 setup 可以多觀察幾秒）

為什麼「早」致命重要：
→ 早 2 小時 vs 晚 2 小時 = 風險回報比可能從 5:1 vs 0.6:1
→ 早進場 = 止損更近（日低離你很近）+ 回報更大
→ 晚進場 = 止損更遠（日低已經離你很遠）+ 回報更小

什麼時候「不猶豫直接買」vs「多觀察幾秒」：
不猶豫 ⚡ → 5星setup / 前一晚仔細分析過 / 機構型 / 突破明確放量
多觀察 🔍 → 3.5-4星 / 有但書 / K線太寬 / 盤前跳空 / 散戶型
```

## 7.4 倉位大小規則

```
最小：帳戶的 10%
最大：帳戶的 25%

5+⭐ → 20-25%
5⭐  → 15-25%
4-4.5⭐ → 10-15%
3-3.5⭐ → ≤ 10%
<3⭐ → 0%（不交易）

永遠不要讓一筆交易決定你的帳戶生死
即使是 5+星 setup 也不超過 25%
```

```
║   止損位 = 當日最低價的下方                                      ║
║   觸發 → 全部賣出                                               ║
║   「You should always be using a stop loss, especially           ║
║    as a beginner and especially with these volatile stocks.」    ║
║                                                                  ║
║   ──────────────────────────────────────────────                ║
║                                                                  ║
║   階段 2：Day 2（進場隔天）                                      ║
║   ────────────────────                                         ║
║   條件：股價仍在你的買入價上方                                   ║
║   行動：將止損移至你的買入價（成本價）                           ║
║   觸發 → 全部賣出                                               ║
║   意義：從此之後最壞情況 = 打平（「free trade」）                ║
║   注意：帖文說「if you wish」→ 建議但非強制                     ║
║         有時止損太緊會被正常波動洗出去                           ║
║         但對初學者來說，移至成本是安全做法                       ║
║                                                                  ║
║   ──────────────────────────────────────────────                ║
║                                                                  ║
║   階段 3：Day 3+（持有期間）                                     ║
║   ────────────────────                                         ║
║   使用 10SMA 作為 Trailing Soft Stop                             ║
║                                                                  ║
║   Soft Stop = 看收盤價，不是盤中價                               ║
║   → 盤中可能暫時跌破 10SMA 又收回                               ║
║   → IF 收盤價即將跌破 10SMA → 賣出全部剩餘持倉                  ║
║                                                                  ║
║   Hard Stop 替代方案：                                           ║
║   → 如果你無法盯盤 → 設硬止損在 10SMA 下方                     ║
║   → 可能被盤中波動誤觸，但至少保護了利潤                        ║
║                                                                  ║
║   ──────────────────────────────────────────────                ║
║                                                                  ║
║   ⚠️ 絕對規則：止損被觸發 = 機械執行，不思考，不猶豫             ║
║   「If it doesn't work, it doesn't work.」                       ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝

時間線摘要：
Day 1：當日低點下方 → 觸發全出（最大容忍風險）
Day 2：成本價      → 觸發全出（最壞打平）
Day 3+：10SMA soft  → 收盤跌破全出（保護利潤，隨趨勢上移）
```

## 12.2 🏆 High Tight Flag — 最重要的單一 Setup

```
╔══════════════════════════════════════════════════════════════════╗
║                                                                  ║
║   「If you could just trade one setup and it would be the        ║
║    high tight flag, you're gonna make so much money you           ║
║    don't even know what to do with it.」                         ║
║                                                                  ║
║   組成要素：                                                     ║
║   ① High：短期內（數週-數月）漲 100%+                           ║
║   ② Tight：極窄回調 < 20-25%，K線越來越小，Higher Lows 清晰     ║
║   ③ Flag：在 10/20SMA 上方整理，均線全部向上                    ║
║                                                                  ║
║   識別標準：                                                     ║
║   □ 前期漲幅 100%+                                              ║
║   □ 回調幅度 < 20-25%                                           ║
║   □ 整理時間 1-5 週（高動量名稱可以更短：3-5 天）               ║
║   □ K 線收緊                                                    ║
║   □ Higher Lows 清晰                                            ║
║   □ 10/20SMA 在整理期間追上來                                   ║
║   □ 均線全部向上                                                ║
║   □ 整理期間成交量萎縮（蓄力）                                  ║
║   □ 圖表乾淨、線性、有秩序                                     ║
║                                                                  ║
║   為什麼這麼強大（供需邏輯）：                                    ║
║   → 巨大漲幅 = 極強需求                                        ║
║   → 極窄回調 = 幾乎沒人賣 = 供給極度稀缺                       ║
║   → 持有者信念極強（不願放手）                                  ║
║   → 一旦新買盤進入（突破）→ 因為沒有賣壓 → 爆發性上漲          ║
║   → 像壓力鍋：水沸騰（High）+ 蓋子緊密（Tight）                ║
║     → 蓋子掀開（突破）= 爆發性蒸汽噴射                         ║
║                                                                  ║
╚══════════════════════════════════════════════════════════════════╝
```

## 12.3 🪜 階梯形態 + 連續突破

```
股票運動方式：漲→整理→漲→整理→漲→整理
每個「整理」= 一個 Flag = 一個獨立的 setup = 一次交易機會
每個「上漲」= 一次突破 = 一次獲利來源

你可以在同一支股票上多次獲利

注意事項：
→ 階梯不會永遠持續（最終某個 flag 會向下崩跌）
→ 每個 flag 都獨立評估（不要因前面成功就假設後面也會）
→ 越後面的台階風險越高（動量可能逐漸衰減）
→ 但只要 setup 品質好（HL + 收緊），仍值得交易
→ 連續突破可以和第一次一樣是 5 星
→ 均線會「降級」：10SMA → 20SMA → 50SMA → 跌破50SMA=結束
```

## 12.4 🎭 假突破後的強化 Setup

```
假突破後的關鍵檢查：
□ 股票回到整理區間內嗎？（是→繼續觀察 / 否=崩跌→放棄）
□ Higher Lows 仍然存在嗎？（是→有效 / 否→結構被破壞）
□ 區間變得更窄嗎？（是→setup更強⭐+ / 否→回到之前品質）
□ 均線仍然向上嗎？（是→趨勢健康 / 否→趨勢轉弱）
□ 20SMA 追上來更多了嗎？（是→更強支撐底部）

IF 全部 ✅ → 重新加入 Watchlist → 下次突破時再次進場
           → 星級可能比上次更高 → 信心更大

遠離 20SMA 的突破更容易假突破：
→ 還沒充分整理 → 20SMA 沒追上來 → 失敗率更高
→ 解決方案：等！等 20SMA 追上來！等區間收緊到極致！
```

## 12.5 📰 Episodic Pivot（事件驅動突破）

```
定義：長期沉睡的股票因重大事件（通常是盈利大幅超預期）而覺醒

關鍵區分：
✅ 好的：突破前 = 長期橫盤（數月-1年+）→ 沉寂越久，爆發越猛
❌ 差的：突破前 = 已經大漲 300%+ → 好消息可能已被反映，失敗率更高

交易方式：
→ 盈利日跳空 < 10% → 可能當天進場
→ 盈利日跳空 > 10% → 通常 pass 當天
→ 等形成第一個整理 flag → 用標準突破策略進場
→ 本質上回到基礎 setup 流程，差別在於催化劑更強
```

## 12.6 🔄 反向拋物線做多

```
定義：極端動量股暴跌後在關鍵長期均線彈跳

必要條件：
① 必須是極端動量股（之前漲幅數百%+）
② 必須有劇烈回調（「straight down, 50-70% in a few days/weeks」）
③ 必須在關鍵均線上找到支撐（50/100/150/200SMA）
④ 必須形成 tight setup → 然後突破

⚠️ 排除條件：
→ 不是極端動量股 → 不適用
→ 回調不劇烈（不是「straight down」）→ 不適用
→ 慢慢陰跌 ≠ 劇烈回調

與標準突破的區別：
→ 標準：上升途中的小整理（順勢）
→ 反向拋物線：暴跌後的底部反彈（逆勢→回歸）
→ 風險更高，是進階技巧，初學者先精通標準突破
```

## 12.7 🐢 50SMA 衝浪 Setup

```
→ 和 20SMA setup 完全一樣的邏輯，只是在 50SMA 上整理
→ 股票速度較慢，回調幅度較大，整理時間較長
→ 小帳戶應優先找 20SMA setup
→ 但仍然是有效的 setup，特別是對較大帳戶
```

## 12.8 📉 崩跌做空（Breakdown Short）

```
→ 突破的鏡像反面：整理 flag 向下崩跌
→ 崩跌日 = 做空進場信號 / 或持有者的離場信號
→ 初學者不要嘗試做空（理論上虧損無上限）
→ 先精通做多的突破策略
```

## 12.9 所有 Setup 的統一底層邏輯

```
每種 setup 都是這三步的變體：

Step 1：識別方向性力量
→ 動量 / 催化事件 / 均線支撐 / 趨勢反轉

Step 2：等待收緊（蓄力）
→ 整理 + Higher Lows + 區間縮窄 → 供需即將失衡

Step 3：突破（釋放）
→ 放量突破 → 進場 → 設止損 → 管理 → 獲利了結

🎯 力量方向 + 收緊蓄力 + 放量突破 = 你的交易機會
```

## 12.10 Setup 快速識別指南

```
🏆 HTF → 「這支漲了超多但就是不肯回調」
🪜 階梯 → 「這看起來像樓梯」
🎭 假突破後強化 → 「被洗了一次但結構更強了」
📰 Episodic Pivot → 「沉睡了很久突然醒了」
🔄 反向拋物線 → 「最強的股票跌到了重要均線上」
🐢 50SMA 衝浪 → 「比較慢但在 50 日線上穩穩地建 HL」
📉 崩跌做空 → 「旗形向下突破了」
```
