# Martin Luk 波段交易系統 — 完整交易流程守則

---

## 一、重點分析與優先排序

按對交易績效的影響程度，由高至低排列：

| 優先級 | 核心領域 | 理由 |
|--------|----------|------|
| **P1** | 風險管理與部位規模 | 決定生存與否的根本——23%勝率仍能獲利全靠此 |
| **P2** | 精準入場技術 | 止損寬度直接決定風險倍數，入場精度與回報呈拋物線關係 |
| **P3** | 出場策略框架 | 決定能否捕捉大趨勢同時保護資本 |
| **P4** | 三重掃描系統 | 機會發現的源頭，決定交易池質量 |
| **P5** | 每日工作流程 | 系統性執行的骨架，確保紀律一致 |
| **P6** | 市場健康評估 | 影響倉位大小、持有時間及策略選擇 |
| **P7** | 心理紀律與自我修正 | 長期存活的保障——報復交易、不守止損等問題 |

## 二、完整交易流程守則

---

### 第一章：核心哲學

> **「我無法預測交易的回報，但我可以限制我的風險。」**

**關鍵統計基準（2024年實績）：**
- 勝率：23%（104勝 / 319負）
- 平均獲利：+15%
- 平均虧損：-3%
- 損益比：5:1
- 數學盈虧平衡點：僅需 17% 勝率

**核心原理：止損寬度與風險倍數的拋物線關係**

| 止損寬度 | 25%獲利交易的R倍數 | 50%獲利交易的R倍數 |
|----------|--------------------|--------------------|
| 3.0% | 8.3R | 16.7R |
| 1.5% | 16.7R（+8R） | 33.3R（+17R） |

**守則：入場精度每提升一個級別，回報呈指數級增長。永遠追求最窄合理止損。**

---

### 第二章：盤前掃描系統（三重掃描器）

#### 通用篩選標準
- ✅ 快速移動股票
- ✅ 處於熱門板塊
- ✅ 平均日內波幅（ADR）> 5%

---

#### 掃描器一：盤前掃描（Pre-Market Scanner）
- **目的：** 識別開盤前的動能——尋找事件驅動型樞紐（Episodic Pivot）機會
- **篩選條件：** 帶顯著成交量跳空高開的股票
- **時間：** 開盤前30分鐘內完成
- **操作：** 記錄至「主要觀察名單」，標記跳空幅度、成交量倍數、所屬板塊

#### 掃描器二：強勢股掃描（Potent Scanner）
- **目的：** 識別板塊輪動與主題性動能
- **篩選條件：** 前一日表現最強的股票
- **關鍵信號：** 同一板塊多隻股票反覆出現 → 資金流入信號，動能往往延續至後續交易日
- **應用實例：** 量子計算主題、加密貨幣關聯動能

#### 掃描器三：領導者掃描（Leader Scanner）
- **目的：** 識別持續相對強勢；兼作市場健康指標
- **篩選條件：** 按月度表現排名（週線級別視角）
- **市場健康判讀：**

| 領導者名單狀態 | 市場含義 | 操作調整 |
|----------------|----------|----------|
| 名單擴張 | 市場強勢 | 增加倉位、延長持有 |
| 名單劇烈收縮 | 市場修正 | 減少倉位、縮短持有、考慮做空 |

---

### 第三章：觀察名單建構與篩選

#### 步驟一：從掃描結果中精選可執行標的

**篩選準則（二選一）：**

**A. 均線支撐/收復型：**
- 價格正在關鍵均線獲得支撐或重新收復
- 關鍵均線：**9 EMA、21 EMA、50 EMA**
- 正在形成底部（Base）

**B. 盤前最強動能型：**
- 盤前交易中表現最佳的標的
- 適用於事件驅動型樞紐交易

#### 步驟二：分類管理

| 分類 | 定義 | 用途 |
|------|------|------|
| **主要觀察名單** | 關鍵股票與即時交易想法 | 核心交易池 |
| **領導股** | 9 EMA > 21 EMA > 50 EMA 的明確上升趨勢 | 做多首選 |
| **平庸股** | 無明確方向偏差 | 暫不交易 |
| **落後股** | 處於下降趨勢 | 做空候選（修正期） |
| **支柱股** | 影響指數的超大型股 | 市場方向參考 |

**市場健康指標：** 當「領導股」類別收縮、「落後股」類別擴張 → 突破策略環境不利，減少交易。

### 第四章：精準入場技術

#### 設置一：前日高點突破（主要設置）

**理想條件清單（越多越好）：**
- [ ] 出現內包日（Inside Day）或極窄幅整理
- [ ] 跳空高開至前日區間上部
- [ ] 突破前日高點
- [ ] 價格與主要均線（9/21/50 EMA）匯聚
- [ ] 在底部區域內形成內包K線或窄幅區域

**入場流程：**
```
日線圖確認格局 → 小時圖定位區域 → 5分鐘圖精確入場
```

#### 設置二：開盤區間高點突破（盤中精準入場）

- **適用：** 事件驅動型樞紐（Episodic Pivot）
- **時間框架：** 1分鐘圖（開盤後立即）或 5分鐘圖（該時間框架建立後）
- **入場：** 價格突破開盤區間高點
- **止損：** 設於開盤區間低點
- **優勢：** 風險參數清晰明確

#### 設置三：盤中區間高點再入場

- **適用：** 開盤區間突破失敗後的二次機會
- **條件：** 股票維持看漲特徵
- **方法：** 利用5分鐘圖在9 EMA上方的整理突破
- **止損：** 5分鐘入場K線低點

#### 入場精度的核心價值

| 入場方式 | 典型止損 | 問題 |
|----------|----------|------|
| 傳統突破入場 | 6-7% | 部位規模受限 |
| **精準入場** | **2-3%** | **相同美元風險下可加大部位** |

**守則：在日線、小時線、5分鐘圖之間頻繁切換，尋找最低風險入場點。**

### 第五章：部位規模與風險管理

#### 5.1 單筆風險規則

| 參數 | 基準值 | 調整規則 |
|------|--------|----------|
| **每筆交易風險** | 帳戶的 **0.5%** | — |
| 回撤期間 | 降低部位 | 「試水溫」模式 |
| 盈利緩衝 + 強勢市場 | 增加曝險 | 利用利潤墊擴張 |

#### 5.2 最大組合曝險

| 股票類型 | 最大倉位比重 |
|----------|-------------|
| 大型流動性股票 | 35% |
| 小型/微型股 | 25-30%（跳空風險較高） |

#### 5.3 止損方法論

**主要止損規則：突破日低點**
- 依據：真正的突破極少回踩突破日低點（經大量圖表研究確認）

**備用止損規則：**
- 若入場點至日低距離 > 5% → 改用 **5分鐘入場K線低點** 作為止損
- 事件驅動型交易中（如GME案例），可使用前一分鐘K線低點

**部位計算公式：**
```
部位規模 = (帳戶淨值 × 0.5%) ÷ (入場價 - 止損價)
```

### 第六章：出場策略框架

#### 6.1 強勢中賣出（Selling into Strength）

**觸發條件（任一出現即考慮減倉）：**
- [ ] 利潤超過 **3倍風險倍數（3R）**
- [ ] 倉位因升值超過 **35%** 組合比重
- [ ] 股票遠離 9 EMA（過度延伸）
- [ ] 對該持倉感到 **亢奮/狂喜**

**執行方式：** 每次減倉 **10-15%**，逐步退出

#### 6.2 弱勢中賣出（Selling into Weakness）

**追蹤止損規則：**

| 情境 | 追蹤均線 | 適用條件 |
|------|----------|----------|
| 標準持倉 | **9 EMA** | 大部分情況 |
| 強勢上升趨勢 + 早期入場 | **21 EMA** | 避免過早退出 |
| 極強趨勢 + 充足利潤墊 | **50 EMA** | 捕捉最大波段 |

#### 6.3 出場策略的自適應調整

| 市場/帳戶狀態 | 傾向 | 理由 |
|---------------|------|------|
| 回撤期間 | **強勢中賣出** | 保護資本優先 |
| 利潤墊充足 + 強勢市場 | **弱勢中追蹤** | 捕捉延伸行情 |

### 第八章：市場適應與進階規則

#### 8.1 權益曲線反饋原則

| 情境 | 解讀 | 行動 |
|------|------|------|
| 你的交易獲利，但大盤掙扎 | 你的系統有效 | 繼續執行，略加謹慎 |
| 大盤強勢，但你的交易虧損 | 你的執行有問題 | **積極重新評估** |

#### 8.2 修正期間的反向操作

- 目標：反彈至下降匯聚均線的股票
- 本質：「反向突破」——相同風險管理原則，方向相反
- 進場：下跌趨勢中的反彈至均線阻力後做空

#### 8.3 關聯性交易（進階）

- **案例：** 透過 Coinbase (COIN) 交易 Bitcoin
- **原理：** 利用相關資產的技術信號作為觸發
- **注意：** 非典型操作，入場規則可能偏離標準系統——需額外謹慎管理風險

### 第九章：心理紀律守則

#### 9.1 三大致命弱點（自我檢查清單）

| 弱點 | 表現 | 防範措施 |
|------|------|----------|
| **報復交易** | 虧損後立即加碼試圖回本 | 虧損日設最大交易次數；強制冷靜期 |
| **虧損時加大風險** | 連虧後提高部位規模 | 連虧後**降低**部位至基準的50% |
| **不遵守止損** | 移動或忽略預設止損 | 入場即設硬止損，不可更改 |

#### 9.2 核心心理原則

1. **進步而非完美：** 即使在三位數回報期間仍多次違反規則——交易在風險管理到位時是容錯的
2. **系統性執行：** 好的交易永遠是通過系統性執行來紀律地維護優勢
3. **盈利緩衝思維：** 利潤提供心理安全邊際，用於在有利時擴張；回撤時收縮
4. **恐慌賣出是最大敵人：** COIN案例——在正常整理中恐慌賣出，錯過105%後續行情

### 第十章：交易案例快速參考

#### GME（2024年5月）— 事件驅動型樞紐

| 項目 | 數值 |
|------|------|
| 入場 | $26.30（1分鐘開盤區間突破） |
| 止損 | $25.47（前一分鐘K線低點，非日低） |
| 風險 | 3.2% |
| 首次減倉 | ~$37（12R，+37%） |
| 最終出場 | ~$56（40R，接近2021阻力） |
| **關鍵教訓** | 日低止損需>6%時，改用更精細的止損點 |

#### COIN — 關聯性交易

| 項目 | 數值 |
|------|------|
| 入場 | $117（VWAP突破，由BTC信號觸發） |
| 止損 | $114.50（2%止損） |
| 出場 | 20R，+40% |
| **關鍵教訓** | 首次交易因恐慌賣出失敗；第二次嚴守規則捕獲全部行情。若守規則，12R減半+餘倉追蹤可獲105% |

#### AMD — 均線匯聚底部突破（2025年，2倍槓桿ETF）

| 項目 | 數值 |
|------|------|
| 入場 | 重新收復關鍵均線後突破前日高/前1小時K線高 |
| 止損 | 日低（均線匯聚點） |
| 持有 | 約9天 |
| 回報 | 35-40% |
| 出場觸發 | 全市場性弱勢（大紅K線日） |
| **關鍵教訓** | 均線匯聚 = 整理結束信號；次日30%跳空高開為持倉中首次經歷 |

### 附錄：執行檢查表（每筆交易前過一遍）

```
入場前檢查：
□ 股票來自三重掃描系統之一？
□ ADR > 5%？
□ 處於熱門板塊？
□ 入場符合三種設置之一？（前日高突破/開盤區間/盤中再入場）
□ 止損距離 ≤ 5%？（否則尋找更精細止損點）
□ 風險 = 帳戶的 0.5%？（根據回撤/盈利調整？）
□ 總曝險未超過上限？（大型35% / 小型25-30%）
□ 市場健康支持做多？（領導者名單狀態？）

持倉中檢查：
□ 達到 3R → 開始減倉 10-15%？
□ 倉位因升值超 35% → 減倉？
□ 追蹤止損設定？（9/21/50 EMA依情境選擇）
□ 感到亢奮 → 減倉信號！
□ 是否有恐慌衝動？→ 檢查出場規則是否已觸發，未觸發則持有

收盤後檢查：
□ 是否偏離了任何規則？記錄下來
□ 觀察名單分類更新？
□ 權益曲線反饋：交易表現 vs 大盤表現？
```
